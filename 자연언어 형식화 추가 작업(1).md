# 작업 단계
남은 추가 작업은 “이론→실험→재현가능성” 관점에서 보면 딱 7개입니다. 지금 코드가 _MVP 파이프라인_까지는 갖췄고, 아래를 하면 “연구/특허/제품” 레벨로 올라갑니다.

## 1) 코퍼스 준비 파이프라인

- 문서 단위 분리(`corpus_docs`) 기준 고정(대화/논문/메모/판례 등)
    
- 문장 분리기(한국어) 붙이기 + 정규화(괄호, 인용, 기호)
    
- “컨텍스트”를 문장 1개로 할지, (문장±1) 윈도우로 할지 확정
    

## 2) 후보/노드 추출 품질 개선(지금은 토크나이저가 너무 거칩니다)

- 형태소/구문 단위 명사구(NP) 추출로 교체(“개념 후보” 품질 상승)
    
- 다의어 노드 분해 규칙(한 노드가 서로 다른 뜻의 컨텍스트를 많이 먹으면 분리)
    

## 3) Lower layer (f)를 “선형+상수”에서 한 단계만 확장

지금은 (x_{t+1}=Ax_t+b)인데, 최소 확장 2안 중 하나는 필요합니다.

- (A) **잔차 MLP 1층**: (x_{t+1}=Ax_t+b+g(x_t)) (작게, 안정성 유지)
    
- (B) **컨텍스트-의존 전이**: `propose_next_contexts`가 아니라 “문서 그래프 전이”로 학습 데이터 생성
    

## 4) Upper layer (\Phi) 업데이트 규칙 검증(핵심 가정 확인)

- 지금은 (W_{t+1}=(1-\eta)W_t+\eta S(x_t)) + prune
    
- 여기서 **S(x)의 정의**(공활성 외에 조건/예외/대립 신호 반영)로 성능이 갈립니다.
    
    - 예: 대립 마커가 있으면 (S)에 “억제 항” 추가
        
    - 예: “즉/정의” 문장에는 “정의 엣지” 가중치 강화
        

## 5) “본질=응축점”을 진짜로 계산하는 코어 추정기로 교체

지금 x0를 평균으로 잡았는데, 본질 가설을 밀려면 최소 하나는 들어가야 합니다.

- 밀도 코어: kNN-density / mean-shift / geometric median
    
- “꼬리 절단”: 중심에서 거리 임계값 밖 컨텍스트를 자동 제외
    

## 6) 평가 지표/벤치마크 만들기

자동 판정은 “건강검진”이고, 성능 평가는 별개입니다.

- 사람 평가: 노드/엣지가 “정의/근거/예외/대립”을 제대로 재현하는지
    
- 내적 지표: 문서 예측(다음 문장/다음 단락), 개념 구분(근접 개념 분리)
    
- 재현성: seed 고정, 코퍼스 버전, 노드 사전 로그 저장
    

## 7) 산출물 포맷 고정(논문/특허/제품용)

- 결과를 **그래프 JSON(노드, 엣지, 가중치, 버킷, 근거 문장)**으로 내보내기
    
- “근거 문장”을 각 엣지에 3~5개 붙여서 설명 가능하게 만들기(설명가능성)

```python
#    geometric median + 꼬리 절단(tail trimming)
# -------------------------

def geometric_median(X: np.ndarray, eps: float = 1e-6, max_iter: int = 200) -> np.ndarray:
    """
    Weiszfeld's algorithm.
    X: (n, d)
    Returns: (d,)
    """
    if X.ndim != 2 or X.shape[0] == 0:
        raise ValueError("X must be (n,d) with n>0")

    y = np.mean(X, axis=0)  # init
    for _ in range(max_iter):
        diff = X - y
        dist = np.linalg.norm(diff, axis=1)
        # avoid division by zero
        mask = dist > eps
        if not np.any(mask):
            return y
        w = 1.0 / np.maximum(dist[mask], eps)
        y_new = np.sum(X[mask] * w[:, None], axis=0) / np.sum(w)
        if np.linalg.norm(y_new - y) <= eps:
            return y_new
        y = y_new
    return y


def estimate_concept_core_x0(
    concept: str,
    corpus_docs: Sequence[Sequence[str]],
    embed,
    *,
    max_contexts: int = 2000,
    trim_quantile: float = 0.90,
    median_eps: float = 1e-6,
    median_max_iter: int = 200,
    seed: int = 7,
) -> Tuple[np.ndarray, Dict[str, Any]]:
    """
    Returns:
      x0_core: (d,)
      info: dict with counts and trimming stats

    Procedure:
      1) collect up to max_contexts sentences containing concept
      2) embed them -> X
      3) compute geometric median m
      4) compute distances ||x_i - m||
      5) keep points within distance quantile q (trim tail)
      6) re-compute geometric median on trimmed set => x0_core
    """
    rng = np.random.default_rng(seed)
    sentences = [s for doc in corpus_docs for s in doc]
    hits = [s for s in sentences if concept in s]
    if not hits:
        raise ValueError(f"No contexts found containing concept='{concept}'.")

    if len(hits) > max_contexts:
        idx = rng.choice(len(hits), size=max_contexts, replace=False)
        hits = [hits[i] for i in idx]

    X = np.stack([embed.encode(s) for s in hits], axis=0).astype(float)
    m = geometric_median(X, eps=median_eps, max_iter=median_max_iter)

    d = np.linalg.norm(X - m[None, :], axis=1)
    q = float(np.quantile(d, trim_quantile))
    keep = d <= q
    X_trim = X[keep]

    if X_trim.shape[0] < max(10, int(0.05 * X.shape[0])):
        # trimming too aggressive; back off
        keep = d <= float(np.quantile(d, min(0.97, trim_quantile + 0.05)))
        X_trim = X[keep]

    x0 = geometric_median(X_trim, eps=median_eps, max_iter=median_max_iter)

    info = {
        "concept": concept,
        "contexts_total_found": len([s for s in sentences if concept in s]),
        "contexts_used": int(X.shape[0]),
        "trim_quantile": float(trim_quantile),
        "trim_threshold_distance": q,
        "kept_after_trim": int(X_trim.shape[0]),
        "d_mean": float(np.mean(d)),
        "d_median": float(np.median(d)),
        "d_q90": float(np.quantile(d, 0.90)),
        "d_q95": float(np.quantile(d, 0.95)),
    }
    return x0, info

def _bucket_of_node(node: str, node_log) -> str:
    # node_log.bucket_nodes: Dict[str, List[str]]
    for b, nodes in node_log.bucket_nodes.items():
        if node in nodes:
            return b
    return "Unknown"


def _top_edges(W: np.ndarray, top_m: int, allow_self: bool = False) -> List[Tuple[int, int, float]]:
    if W.size == 0:
        return []
    flat = W.ravel()
    order = np.argsort(flat)[::-1]
    out = []
    for idx in order:
        w = float(flat[idx])
        if w <= 0:
            break
        i = int(idx // W.shape[1])
        j = int(idx % W.shape[1])
        if (not allow_self) and (i == j):
            continue
        out.append((i, j, w))
        if len(out) >= top_m:
            break
    return out


def export_graph_json_with_evidence(
    art,
    corpus_docs: Sequence[Sequence[str]],
    embed,
    *,
    top_edges: int = 200,
    evidence_per_edge: int = 4,
    evidence_pool: int = 4000,
    seed: int = 7,
) -> Dict[str, Any]:
    """
    Produces a JSON-serializable dict.

    Evidence policy (MVP, deterministic-ish):
      - For each sentence s, compute a_s = upper.a(embed(s))
      - For an edge i->j, score_s = a_s[i] * a_s[j]
      - Evidence = top sentences by score_s among a pool
    """
    rng = np.random.default_rng(seed)

    concept_nodes = art.concept_nodes
    node_index = art.node_index
    W = art.result.W_star
    a_star = np.asarray(art.upper.a(art.result.x_star), dtype=float)

    # flatten sentences and sample a pool (for speed)
    sentences_all = [s for doc in corpus_docs for s in doc]
    if len(sentences_all) > evidence_pool:
        idx = rng.choice(len(sentences_all), size=evidence_pool, replace=False)
        sentences = [sentences_all[i] for i in idx]
    else:
        sentences = sentences_all

    # precompute sentence embeddings + activations once
    Xs = np.stack([embed.encode(s) for s in sentences], axis=0).astype(float)
    As = np.stack([art.upper.a(x) for x in Xs], axis=0).astype(float)  # (M, N)

    # nodes
    nodes_json = []
    for k, name in enumerate(concept_nodes):
        nodes_json.append({
            "id": k,
            "name": name,
            "bucket": _bucket_of_node(name, art.node_log),
            "a_at_x_star": float(a_star[k]) if k < a_star.shape[0] else None,
        })

    # edges + evidence
    edge_triples = _top_edges(W, top_m=top_edges, allow_self=False)

    edges_json = []
    for i, j, w in edge_triples:
        # evidence scoring
        scores = As[:, i] * As[:, j]  # (M,)
        topk = np.argsort(scores)[::-1][:evidence_per_edge]

        evidences = []
        for t in topk:
            evidences.append({
                "sentence": sentences[int(t)],
                "score": float(scores[int(t)]),
                "a_i": float(As[int(t), i]),
                "a_j": float(As[int(t), j]),
            })

        edges_json.append({
            "source": int(i),
            "target": int(j),
            "weight": float(w),
            "evidence": evidences,
        })

    out = {
        "meta": {
            "concept": art.concept,
            "N_nodes": len(concept_nodes),
            "converged": bool(art.result.converged),
            "steps": int(art.result.steps),
            "dx_last": float(art.result.history["dx"][-1]) if art.result.history.get("dx") else None,
            "dW_last": float(art.result.history["dW"][-1]) if art.result.history.get("dW") else None,
            "W_mass": float(np.sum(W)) if W.size else 0.0,
            "A_spectral_norm": float(spectral_norm(art.lower.A)),
            "evidence_pool_size": len(sentences),
            "params": {
                "eta": float(art.upper.eta),
                "beta": float(art.upper.beta),
                "prune_theta": float(art.upper.prune_theta),
            },
        },
        "nodes": nodes_json,
        "edges": edges_json,
    }
    return out
```

```python
_REL_PATTERNS: Dict[str, List[re.Pattern]] = {
    # "definition / rephrase"
    "definition": [
        re.compile(r"\b(즉|다시\s*말해|말하자면|바로)\b"),
        re.compile(r"\b(~?란|~?이란)\b"),  # very rough
        re.compile(r"\b(정의|의미)\b"),
        re.compile(r"라고\s*(한다|부른다|정의한다)"),

    # "justification / reason"
    "justification": [
        re.compile(r"\b(왜냐하면|이유는|근거는)\b"),
        re.compile(r"\b(따라서|그러므로|그러니|그래서)\b"),

# Priority order when multiple match
_REL_PRIORITY = ["definition", "condition_exception", "contrast", "justification", "causal", "example"]


def detect_relation_hints(sentence: str) -> List[str]:
    """
    Returns list of relation labels detected in the sentence.
    """
    hits = []
    s = sentence
    for label, pats in _REL_PATTERNS.items():
        for p in pats:
            if p.search(s):
                hits.append(label)
                break
    # sort by priority, keep order
    hits_sorted = [lab for lab in _REL_PRIORITY if lab in set(hits)]
    # include non-priority labels (if any) after
    for lab in hits:
        if lab not in hits_sorted:
            hits_sorted.append(lab)
    return hits_sorted


def pick_primary_relation(hints: List[str]) -> Optional[str]:
    if not hints:
        return None
    return hints[0]

def export_graph_json_with_evidence_and_relation_hints(
    art: "PipelineArtifacts",
    corpus_docs: Sequence[Sequence[str]],
    embed,
    *,
    top_edges: int = 200,

    seed: int = 7,

    relation_boost: float = 1.10,

    Same as export_graph_json_with_evidence, but:
      - each evidence sentence has relation_hints: [..]
      - edge includes relation_hint_stats: histogram over hints
      - optionally boosts evidence score when relation hints exist

    Evidence score:
      score = a_i * a_j
      if hints exist: score *= relation_boost
    """

    concept_nodes = art.concept_nodes
    W = art.result.W_star
    a_star = np.asarray(art.upper.a(art.result.x_star), dtype=float)

    sentences_all = [s for doc in corpus_docs for s in doc]
    if restrict_evidence_to_concept_contexts:
        sentences_all = [s for s in sentences_all if art.concept in s]

        idx = rng.choice(len(sentences_all), size=evidence_pool, replace=False)
        sentences = [sentences_all[i] for i in idx]
    else:
        sentences = sentences_all

    # Precompute embeddings + activations

    As = np.stack([art.upper.a(x) for x in Xs], axis=0).astype(float)  # (M, N)

    # Precompute relation hints for pool sentences
    hint_lists: List[List[str]] = [detect_relation_hints(s) for s in sentences]
    hint_primary: List[Optional[str]] = [pick_primary_relation(h) for h in hint_lists]
    hint_has_any = np.array([1.0 if h else 0.0 for h in hint_lists], dtype=float)

            if node in nodes:
                return b
        return "Unknown"

    for k, name in enumerate(concept_nodes):
        nodes_json.append({
            "id": k,
            "name": name,
            "bucket": _bucket_of_node(name),
            "a_at_x_star": float(a_star[k]) if k < a_star.shape[0] else None,
        })

    # Edge selection helper
    def _top_edges(Wm: np.ndarray, top_m: int) -> List[Tuple[int, int, float]]:

            return []
        flat = Wm.ravel()

        out = []

            if w <= 0:

            i = int(idx_ // Wm.shape[1])
            j = int(idx_ % Wm.shape[1])
            if i == j:
                continue
            out.append((i, j, w))
            if len(out) >= top_m:
                break
        return out

    edge_triples = _top_edges(W, top_edges)

    edges_json = []
    for i, j, w in edge_triples:
        base = As[:, i] * As[:, j]  # (M,)
        # slight boost if any relation hint exists
        boosted = base * (1.0 + (relation_boost - 1.0) * hint_has_any)

        topk_idx = np.argsort(boosted)[::-1][:evidence_per_edge]

        evidences = []
        hint_hist: Dict[str, int] = {}
        for t in topk_idx:
            t = int(t)
            rels = hint_lists[t]
            primary = hint_primary[t]
            if primary is not None:
                hint_hist[primary] = hint_hist.get(primary, 0) + 1

            evidences.append({
                "sentence": sentences[t],
                "score": float(boosted[t]),
                "base_score": float(base[t]),
                "a_i": float(As[t, i]),
                "a_j": float(As[t, j]),
                "relation_hints": rels,
                "primary_relation_hint": primary,
            })

        edges_json.append({
            "source": int(i),
            "target": int(j),
            "weight": float(w),
            "relation_hint_stats": hint_hist,  # coarse label histogram from top evidence
            "evidence": evidences,
        })

    out = {
        "meta": {
            "concept": art.concept,
            "N_nodes": len(concept_nodes),
            "converged": bool(art.result.converged),
            "steps": int(art.result.steps),
            "dx_last": float(art.result.history["dx"][-1]) if art.result.history.get("dx") else None,
            "dW_last": float(art.result.history["dW"][-1]) if art.result.history.get("dW") else None,
            "W_mass": float(np.sum(W)) if W.size else 0.0,
            "A_spectral_norm": float(spectral_norm(art.lower.A)),
            "evidence_pool_size": len(sentences),
            "evidence_per_edge": int(evidence_per_edge),
            "relation_boost": float(relation_boost),
            "restrict_evidence_to_concept_contexts": bool(restrict_evidence_to_concept_contexts),
            "params": {
                "eta": float(art.upper.eta),
                "beta": float(art.upper.beta),
                "prune_theta": float(art.upper.prune_theta),
            },
        },
        "nodes": nodes_json,
        "edges": edges_json,
    }
    return out
```

### 무엇이 달라졌는지(핵심만)

- 각 엣지의 evidence 문장에 `relation_hints`, `primary_relation_hint`가 붙습니다.
    
- 엣지 자체에도 `relation_hint_stats`(상위 evidence들에서 어떤 관계 라벨이 많이 나왔는지 히스토그램)가 붙습니다.

```python
# -------------------------
# Edge-type promotion rules

def promote_edge_type_from_evidence(
    relation_hint_stats: Dict[str, int],
    *,
    min_votes: int = 2,
    dominance_ratio: float = 0.60,
    priority: Optional[List[str]] = None,
) -> Tuple[str, Dict[str, Any]]:
    """
    Decide edge_type based on relation_hint_stats (histogram from top evidence).

    Rule:
      - If total labeled votes < min_votes => "unsure"
      - Else pick label with max votes.
      - If max_votes / total_votes >= dominance_ratio => that label
      - Else => "mixed"
      - Tie-break uses priority order.

    Returns:
      (edge_type, decision_info)
    """
    priority = priority or ["definition", "condition_exception", "contrast", "justification", "causal", "example"]

    # total labeled votes
    total = int(sum(relation_hint_stats.values()))
    if total < min_votes:
        return "unsure", {
            "total_votes": total,
            "reason": "insufficient_labeled_votes",
            "min_votes": min_votes,
        }

    # find max vote labels
    maxv = max(relation_hint_stats.values())
    top_labels = [k for k, v in relation_hint_stats.items() if v == maxv]

    # tie-break by priority
    chosen = None
    for p in priority:
        if p in top_labels:
            chosen = p
            break
    if chosen is None:
        chosen = sorted(top_labels)[0]

    ratio = float(maxv / total) if total > 0 else 0.0

    if ratio >= dominance_ratio:
        return chosen, {
            "total_votes": total,
            "chosen": chosen,
            "max_votes": int(maxv),
            "dominance_ratio": ratio,
            "threshold": dominance_ratio,
            "reason": "dominant_label",
        }

    return "mixed", {
        "total_votes": total,
        "chosen": chosen,
        "max_votes": int(maxv),
        "dominance_ratio": ratio,
        "threshold": dominance_ratio,
        "reason": "no_dominant_label",
    }


def promote_edge_types_in_graph(
    graph_obj: Dict[str, Any],
    *,
    min_votes: int = 2,
    dominance_ratio: float = 0.60,
    priority: Optional[List[str]] = None,
    write_back_decision_info: bool = True,
) -> Dict[str, Any]:
    """
    Adds:
      edge["edge_type"] = {definition|condition_exception|contrast|justification|causal|example|mixed|unsure}
      edge["edge_type_decision"] (optional)

    Returns a modified copy (does not mutate input).
    """
    priority = priority or ["definition", "condition_exception", "contrast", "justification", "causal", "example"]

    out = {
        **graph_obj,
        "edges": [],
    }

    for e in graph_obj.get("edges", []):
        stats = e.get("relation_hint_stats", {}) or {}

            stats,
            min_votes=min_votes,
            dominance_ratio=dominance_ratio,
            priority=priority,

        e2 = dict(e)
        e2["edge_type"] = edge_type
        if write_back_decision_info:
            e2["edge_type_decision"] = info
        out["edges"].append(e2)

    # Record promotion settings in meta
    meta = dict(out.get("meta", {}))
    meta["edge_type_promotion"] = {
        "min_votes": int(min_votes),
        "dominance_ratio": float(dominance_ratio),
        "priority": priority,
        "write_back_decision_info": bool(write_back_decision_info),
    }
    out["meta"] = meta

    return out
```

### 승격 규칙이 의미하는 바(정확히)

- 각 엣지는 상위 evidence 문장들에서 나온 `primary_relation_hint`들을 집계해서 `relation_hint_stats`를 갖고 있습니다.
    
- 그 히스토그램으로 투표합니다.
    
    - 라벨 투표 수 합이 `min_votes` 미만이면 `edge_type="unsure"`
        
    - 최다 득표 라벨이 전체의 `dominance_ratio` 이상이면 그 라벨로 확정
        
    - 아니면 `edge_type="mixed"`로 둡니다(애매함을 그대로 보존)
        

원하시면, `mixed/unsure`인 엣지만 따로 모아서 **추가 evidence를 더 긁어오거나(evidence_per_edge 확장), 혹은 패턴 규칙을 더 강하게 적용해서 재분류**하는 “2차 판정기”도 바로 붙여드릴 수 있습니다.

```python
# -------------------------
# Utility: get top evidence for a given edge (i->j) from a sentence pool
# -------------------------

def _score_edge_evidence_pool(
    i: int,
    j: int,
    sentences: List[str],
    As: np.ndarray,  # (M, N) activations per sentence
    hint_lists: List[List[str]],
    *,
    evidence_count: int,
    relation_boost: float,
) -> Tuple[List[Dict[str, Any]], Dict[str, int]]:
    """
    Returns:
      evidences: list of evidence dicts
      hint_hist: histogram of primary relation hints among selected evidence
    """
    base = As[:, i] * As[:, j]
    has_any = np.array([1.0 if h else 0.0 for h in hint_lists], dtype=float)
    score = base * (1.0 + (relation_boost - 1.0) * has_any)

    topk = np.argsort(score)[::-1][:evidence_count]

    evidences: List[Dict[str, Any]] = []
    hint_hist: Dict[str, int] = {}

    for t in topk:
        t = int(t)
        rels = hint_lists[t]
        primary = pick_primary_relation(rels)
        if primary is not None:
            hint_hist[primary] = hint_hist.get(primary, 0) + 1

        evidences.append({
            "sentence": sentences[t],
            "score": float(score[t]),
            "base_score": float(base[t]),
            "a_i": float(As[t, i]),
            "a_j": float(As[t, j]),
            "relation_hints": rels,
            "primary_relation_hint": primary,
        })
    return evidences, hint_hist

def second_stage_edge_type_adjudication(
    art,
    graph_obj: Dict[str, Any],
    corpus_docs: Sequence[Sequence[str]],
    embed,
    *,
    # when to trigger 2nd stage
    target_edge_types: Tuple[str, ...] = ("mixed", "unsure"),
    # expand evidence to stabilize votes
    expanded_evidence_per_edge: int = 12,
    evidence_pool: int = 12000,
    seed: int = 7,
    relation_boost: float = 1.10,
    restrict_evidence_to_concept_contexts: bool = False,

    # promotion parameters (stricter than stage-1)
    min_votes: int = 4,
    dominance_ratio: float = 0.65,
    priority: Optional[List[str]] = None,

    # extra: pattern override — if many evidence sentences contain explicit marker of a label, force it
    marker_override_min_hits: int = 4,
) -> Dict[str, Any]:
    """
    Second-stage:
      - For edges with edge_type in target_edge_types:
        - Re-score with a larger evidence pool + more evidence sentences
        - Recompute relation_hint_stats (from expanded evidence)
        - Re-run promotion with stricter thresholds
        - Optional marker override: if one label appears >= marker_override_min_hits in stats, force it.

    Returns modified copy of graph_obj with:
      edge["edge_type_stage2"], edge["edge_type_decision_stage2"], edge["evidence_stage2"] (optional)
      and optionally updates edge_type if stage2 is confident.
    """
    rng = np.random.default_rng(seed)
    priority = priority or ["definition", "condition_exception", "contrast", "justification", "causal", "example"]

    # Flatten and sample pool
    sentences_all = [s for doc in corpus_docs for s in doc]
    if restrict_evidence_to_concept_contexts:
        sentences_all = [s for s in sentences_all if art.concept in s]
    if not sentences_all:
        raise ValueError("No sentences available for stage-2 evidence pool.")

    if len(sentences_all) > evidence_pool:
        idx = rng.choice(len(sentences_all), size=evidence_pool, replace=False)
        sentences = [sentences_all[i] for i in idx]
    else:
        sentences = sentences_all

    # Precompute embeddings + activations + hints
    Xs = np.stack([embed.encode(s) for s in sentences], axis=0).astype(float)
    As = np.stack([art.upper.a(x) for x in Xs], axis=0).astype(float)  # (M,N)
    hint_lists: List[List[str]] = [detect_relation_hints(s) for s in sentences]

    # Build output graph copy
    out = {**graph_obj, "edges": []}
    meta = dict(out.get("meta", {}))
    meta["edge_type_stage2"] = {
        "expanded_evidence_per_edge": int(expanded_evidence_per_edge),
        "evidence_pool": int(len(sentences)),
        "min_votes": int(min_votes),
        "dominance_ratio": float(dominance_ratio),
        "priority": priority,
        "marker_override_min_hits": int(marker_override_min_hits),
        "relation_boost": float(relation_boost),
        "restrict_evidence_to_concept_contexts": bool(restrict_evidence_to_concept_contexts),
        "targets": list(target_edge_types),
    }
    out["meta"] = meta

    for e in graph_obj.get("edges", []):
        e2 = dict(e)
        et = e.get("edge_type", "unsure")

        if et not in target_edge_types:
            out["edges"].append(e2)
            continue

        i = int(e["source"])
        j = int(e["target"])

        evid2, hist2 = _score_edge_evidence_pool(
            i=i, j=j,
            sentences=sentences,
            As=As,
            hint_lists=hint_lists,
            evidence_count=expanded_evidence_per_edge,
            relation_boost=relation_boost,
        )

        # Stage-2 promotion
        edge_type2, info2 = promote_edge_type_from_evidence(
            hist2,
            min_votes=min_votes,
            dominance_ratio=dominance_ratio,
            priority=priority,
        )

        # Optional marker override: if one label appears many times, force it
        forced = None
        if hist2:
            best_label = max(hist2.items(), key=lambda kv: kv[1])[0]
            if hist2.get(best_label, 0) >= marker_override_min_hits:
                forced = best_label

        if forced is not None and edge_type2 in ("mixed", "unsure"):
            edge_type2 = forced
            info2 = {
                **info2,
                "reason": "marker_override",
                "forced_label": forced,
                "forced_hits": int(hist2.get(forced, 0)),
            }

        e2["relation_hint_stats_stage2"] = hist2
        e2["edge_type_stage2"] = edge_type2
        e2["edge_type_decision_stage2"] = info2
        e2["evidence_stage2"] = evid2  # keep separate so you still have stage-1 evidence

        # If stage2 is confident (not mixed/unsure), overwrite main edge_type
        if edge_type2 not in ("mixed", "unsure"):
            e2["edge_type_prev"] = e.get("edge_type", None)
            e2["edge_type"] = edge_type2
            e2["edge_type_decision"] = info2

        out["edges"].append(e2)

    return out
```

### 2차 판정기가 하는 일(정확히)

- `mixed/unsure` 엣지만 골라서 **증거를 더 많이(예: 4→12개)** 모읍니다.
    
- 표본도 더 큽니다(**evidence_pool 4k→12k**).
    
- 그 증거들로 투표를 다시 해서, **지배적 라벨이 생기면 확정**합니다.
    
- 그래도 애매하면 `mixed/unsure` 유지하되, `edge_type_stage2`와 `relation_hint_stats_stage2`, `evidence_stage2`를 남겨서 디버깅 가능하게 합니다.
    
- 확정되면 `edge_type`을 실제로 업데이트하고, 이전 값은 `edge_type_prev`로 보존합니다.
    

원하시면 다음 단계로, stage2에서도 애매한 엣지에 대해 **노드(i/j) 주변 서브그래프를 보고(삼각 폐쇄, 공통 이웃의 라벨)** 라벨을 정하는 _그래프 기반 3차 판정기_까지 붙일 수 있습니다.

# 그래프 기반 3차 판정기
```python
from __future__ import annotations

from typing import Dict, List, Tuple, Any, Optional
import math


# -------------------------
# 3rd-stage (graph-structure) adjudicator
# - Uses triadic closure + neighborhood label consistency
# -------------------------

_CONFIDENT_TYPES = {"definition", "condition_exception", "contrast", "justification", "causal", "example"}
_AMBIG_TYPES = {"mixed", "unsure"}


def _edge_confidence_from_decision(decision: Optional[Dict[str, Any]]) -> float:
    """
    Convert decision metadata into a soft confidence in [0,1].
    Works with stage1/stage2 decision payloads produced earlier.

    Heuristic:
      - dominant_label with high dominance_ratio -> high confidence
      - marker_override -> medium-high
      - otherwise -> low
    """
    if not isinstance(decision, dict):
        return 0.35
    reason = decision.get("reason", "")
    dom = float(decision.get("dominance_ratio", 0.0) or 0.0)
    total = float(decision.get("total_votes", 0.0) or 0.0)

    if reason == "dominant_label":
        # dom in [threshold, 1]
        return max(0.55, min(0.98, 0.35 + 0.75 * dom))
    if reason == "marker_override":
        # depends on forced hits
        hits = float(decision.get("forced_hits", 0.0) or 0.0)
        return max(0.55, min(0.90, 0.55 + 0.03 * hits))
    if reason == "insufficient_labeled_votes":
        return 0.30

    # generic fallback
    if total >= 4 and dom >= 0.5:
        return 0.55
    return 0.40


def _get_best_decision_payload(e: Dict[str, Any]) -> Optional[Dict[str, Any]]:
    """
    Prefer stage2 decision if present, else stage1.
    """
    if "edge_type_decision_stage2" in e and isinstance(e["edge_type_decision_stage2"], dict):
        return e["edge_type_decision_stage2"]
    if "edge_type_decision" in e and isinstance(e["edge_type_decision"], dict):
        return e["edge_type_decision"]
    return None


def _build_adjacency(graph_obj: Dict[str, Any]) -> Tuple[Dict[int, List[Tuple[int, Dict[str, Any]]]],
                                                        Dict[int, List[Tuple[int, Dict[str, Any]]]]]:
    """
    Returns:
      out_adj[u] = list of (v, edge_dict)
      in_adj[v]  = list of (u, edge_dict)
    """
    out_adj: Dict[int, List[Tuple[int, Dict[str, Any]]]] = {}
    in_adj: Dict[int, List[Tuple[int, Dict[str, Any]]]] = {}
    for e in graph_obj.get("edges", []):
        u = int(e["source"])
        v = int(e["target"])
        out_adj.setdefault(u, []).append((v, e))
        in_adj.setdefault(v, []).append((u, e))
    return out_adj, in_adj


def _edge_weight(e: Dict[str, Any]) -> float:
    w = e.get("weight", 0.0)
    try:
        return float(w)
    except Exception:
        return 0.0


def _type_of_edge(e: Dict[str, Any]) -> str:
    # after stage2, main edge_type may have been overwritten; trust e["edge_type"]
    t = e.get("edge_type", "unsure")
    return str(t)


def _triadic_vote_scores(
    i: int,
    j: int,
    out_adj: Dict[int, List[Tuple[int, Dict[str, Any]]]],
    in_adj: Dict[int, List[Tuple[int, Dict[str, Any]]]],
    *,
    max_k: int = 50,
    weight_power: float = 1.0,
    conf_power: float = 1.0,
) -> Dict[str, float]:
    """
    Score labels using two-hop paths i -> k -> j.

    For each k:
      score += (w_ik * w_kj)^weight_power * (conf_ik * conf_kj)^conf_power
      added to label L if types(ik)==types(kj)==L and L is confident.

    Additionally:
      if types differ but both confident, add smaller support to both (soft).
    """
    scores: Dict[str, float] = {t: 0.0 for t in _CONFIDENT_TYPES}

    out_i = out_adj.get(i, [])
    in_j = in_adj.get(j, [])

    # map i->k
    out_map: Dict[int, Dict[str, Any]] = {k: e for (k, e) in out_i}
    # list candidate k that also go to j (k->j exists)
    # We can find via in_j sources
    candidates = [k for (k, _) in in_j if k in out_map]

    # limit to top-k by product weight (fast, stable)
    def key_fn(k: int) -> float:
        e1 = out_map[k]
        # find k->j edge
        e2 = None
        for (src, ee) in in_j:
            if src == k:
                e2 = ee
                break
        if e2 is None:
            return 0.0
        return _edge_weight(e1) * _edge_weight(e2)

    candidates.sort(key=key_fn, reverse=True)
    candidates = candidates[:max_k]

    # build quick lookup for k->j edge
    in_j_map: Dict[int, Dict[str, Any]] = {k: e for (k, e) in in_j}

    for k in candidates:
        e_ik = out_map[k]
        e_kj = in_j_map[k]

        t1 = _type_of_edge(e_ik)
        t2 = _type_of_edge(e_kj)

        if t1 not in _CONFIDENT_TYPES or t2 not in _CONFIDENT_TYPES:
            continue

        w = (_edge_weight(e_ik) * _edge_weight(e_kj)) ** weight_power
        c1 = _edge_confidence_from_decision(_get_best_decision_payload(e_ik))
        c2 = _edge_confidence_from_decision(_get_best_decision_payload(e_kj))
        c = (c1 * c2) ** conf_power
        base = w * c

        if t1 == t2:
            scores[t1] += base
        else:
            # soft support: the path suggests "relatedness", but relation type is unclear
            scores[t1] += 0.35 * base
            scores[t2] += 0.35 * base

    return scores


def _neighborhood_consistency_scores(
    i: int,
    j: int,
    out_adj: Dict[int, List[Tuple[int, Dict[str, Any]]]],
    in_adj: Dict[int, List[Tuple[int, Dict[str, Any]]]],
    *,
    take_top: int = 25,
    weight_power: float = 1.0,
    conf_power: float = 1.0,
) -> Dict[str, float]:
    """
    Additional signal:
      - outgoing label distribution of node i
      - incoming label distribution of node j

    Score(L) = sum_out_i(L) + sum_in_j(L) where each edge contributes:
      (weight^weight_power) * (conf^conf_power)
    """
    scores: Dict[str, float] = {t: 0.0 for t in _CONFIDENT_TYPES}

    def add_edges(edge_list: List[Tuple[int, Dict[str, Any]]]):
        # choose top by weight to reduce noise
        edge_list2 = sorted(edge_list, key=lambda kv: _edge_weight(kv[1]), reverse=True)[:take_top]
        for _, e in edge_list2:
            t = _type_of_edge(e)
            if t not in _CONFIDENT_TYPES:
                continue
            w = _edge_weight(e) ** weight_power
            c = _edge_confidence_from_decision(_get_best_decision_payload(e)) ** conf_power
            scores[t] += w * c

    add_edges(out_adj.get(i, []))
    add_edges(in_adj.get(j, []))
    return scores


def _combine_scores(
    tri: Dict[str, float],
    nei: Dict[str, float],
    *,
    alpha_triad: float = 0.70,
    alpha_nei: float = 0.30,
) -> Dict[str, float]:
    out: Dict[str, float] = {}
    for t in _CONFIDENT_TYPES:
        out[t] = alpha_triad * float(tri.get(t, 0.0)) + alpha_nei * float(nei.get(t, 0.0))
    return out


def _promote_from_scores(
    scores: Dict[str, float],
    *,
    min_total_score: float = 1e-6,
    dominance_ratio: float = 0.60,
    margin_ratio: float = 0.15,
    priority: Optional[List[str]] = None,
) -> Tuple[str, Dict[str, Any]]:
    """
    Decide type from continuous scores.

    Rule:
      - If sum(scores) < min_total_score -> "unsure"
      - pick best label (tie-break by priority)
      - require best / total >= dominance_ratio
      - and (best - second)/best >= margin_ratio to avoid fragile picks
    """
    priority = priority or ["definition", "condition_exception", "contrast", "justification", "causal", "example"]

    total = float(sum(scores.values()))
    if total < min_total_score:
        return "unsure", {"reason": "insufficient_graph_signal", "total_score": total}

    # best, second
    items = list(scores.items())
    items.sort(key=lambda kv: kv[1], reverse=True)

    best_label, best = items[0]
    second = items[1][1] if len(items) > 1 else 0.0

    # tie-break by priority if close
    tied = [lab for lab, sc in items if abs(sc - best) <= 1e-12]
    if len(tied) > 1:
        for p in priority:
            if p in tied:
                best_label = p
                break

    dom = best / total if total > 0 else 0.0
    margin = (best - second) / max(best, 1e-12)

    if dom >= dominance_ratio and margin >= margin_ratio:
        return best_label, {
            "reason": "graph_dominant",
            "total_score": total,
            "best": best,
            "second": second,
            "dominance_ratio": dom,
            "margin_ratio": margin,
            "thresholds": {"dominance_ratio": dominance_ratio, "margin_ratio": margin_ratio},
            "scores": scores,
        }

    return "mixed", {
        "reason": "graph_ambiguous",
        "total_score": total,
        "best_label": best_label,
        "best": best,
        "second": second,
        "dominance_ratio": dom,
        "margin_ratio": margin,
        "thresholds": {"dominance_ratio": dominance_ratio, "margin_ratio": margin_ratio},
        "scores": scores,
    }


def third_stage_edge_type_adjudication_graph_based(
    graph_obj: Dict[str, Any],
    *,
    target_edge_types: Tuple[str, ...] = ("mixed", "unsure"),
    # triadic closure params
    max_k: int = 50,
    # combination weights
    alpha_triad: float = 0.70,
    alpha_nei: float = 0.30,
    # decision thresholds
    dominance_ratio: float = 0.62,
    margin_ratio: float = 0.18,
    min_total_score: float = 1e-6,
    priority: Optional[List[str]] = None,
    # if stage3 is confident, overwrite edge_type
    overwrite_when_confident: bool = True,
) -> Dict[str, Any]:
    """
    Stage-3 adjudicator:
      - Uses ONLY the graph structure and already-classified edges.
      - For edges still mixed/unsure, estimates type using:
          (i -> k -> j) two-hop consistency + neighborhood label bias.

    Adds per-edge:
      - edge_type_stage3
      - edge_type_decision_stage3
      - (optional) overwrites edge_type when confident
    """
    priority = priority or ["definition", "condition_exception", "contrast", "justification", "causal", "example"]

    out_adj, in_adj = _build_adjacency(graph_obj)

    out = {**graph_obj, "edges": []}
    meta = dict(out.get("meta", {}))
    meta["edge_type_stage3"] = {
        "target_edge_types": list(target_edge_types),
        "max_k": int(max_k),
        "alpha_triad": float(alpha_triad),
        "alpha_nei": float(alpha_nei),
        "dominance_ratio": float(dominance_ratio),
        "margin_ratio": float(margin_ratio),
        "min_total_score": float(min_total_score),
        "priority": priority,
        "overwrite_when_confident": bool(overwrite_when_confident),
    }
    out["meta"] = meta

    for e in graph_obj.get("edges", []):
        e2 = dict(e)
        et = str(e.get("edge_type", "unsure"))

        if et not in target_edge_types:
            out["edges"].append(e2)
            continue

        i = int(e["source"])
        j = int(e["target"])

        tri = _triadic_vote_scores(i, j, out_adj, in_adj, max_k=max_k)
        nei = _neighborhood_consistency_scores(i, j, out_adj, in_adj)
        comb = _combine_scores(tri, nei, alpha_triad=alpha_triad, alpha_nei=alpha_nei)

        edge_type3, info3 = _promote_from_scores(
            comb,
            min_total_score=min_total_score,
            dominance_ratio=dominance_ratio,
            margin_ratio=margin_ratio,
            priority=priority,
        )

        e2["edge_type_stage3"] = edge_type3
        e2["edge_type_decision_stage3"] = info3

        if overwrite_when_confident and edge_type3 in _CONFIDENT_TYPES:
            e2["edge_type_prev"] = e2.get("edge_type_prev", e.get("edge_type", None))
            e2["edge_type"] = edge_type3
            # also overwrite canonical decision so downstream uses it
            e2["edge_type_decision"] = info3

        out["edges"].append(e2)

    return out
```

### 3차 판정기가 “무엇을 근거로” 확정하는지

- **삼각 폐쇄( i→k→j )**에서, 양쪽 엣지가 같은 타입이면 그 타입에 강하게 점수 부여합니다.
    
- **주변 분포( i의 outgoing, j의 incoming )**에서 특정 타입이 일관되게 강하면 그 방향으로 보조 점수 부여합니다.
    
- 점수 기반으로
    
    - 최상위 타입이 **총점 대비 비율(dominance)**을 넘고
        
    - 2등과의 **격차(margin)**도 충분하면
        
    - `edge_type_stage3`로 확정하고(옵션에 따라) `edge_type`을 덮어씁니다.
        

원하시면 다음은 더 “정교한 4차”도 가능합니다. 예를 들면,

- 타입별 전이 규칙(정의 다음엔 예외/조건이 자주 온다 같은) 반영,
    
- 노드 버킷(C: 대립/경계 등)과 타입의 합치성 제약,
    
- 또는 edge evidence의 규칙 기반 강제(“~란” 문장 있으면 definition 강제) 같은 것입니다.

# 4차 판정기
```python
from __future__ import annotations

from typing import Dict, List, Tuple, Any, Optional, Set
import math
import re

# Assumes you already have (from prior steps):
# - _CONFIDENT_TYPES, _AMBIG_TYPES
# - detect_relation_hints, pick_primary_relation
# - _build_adjacency, _type_of_edge, _edge_weight
# - _edge_confidence_from_decision, _get_best_decision_payload
# - third_stage_edge_type_adjudication_graph_based (optional; stage4 can run after it)
#
# And node buckets exist in graph_obj["nodes"][k]["bucket"].


# -------------------------
# 4th-stage: constraint-aware adjudicator
#   Signals combined:
#   (A) hard evidence rules (marker -> forced type)
#   (B) bucket compatibility constraints (soft/hard)
#   (C) local transition priors (edge_type sequences around endpoints)
#   (D) graph scores from stage3 decision payload (if present) or reconstructable via triadic methods
# -------------------------

_DEFAULT_PRIORITY = ["definition", "condition_exception", "contrast", "justification", "causal", "example"]

# (A) Stronger hard rules than stage1/2 label hints:
_HARD_MARKERS = {
    "definition": [
        re.compile(r"\b(즉|다시\s*말해|말하자면|바로)\b"),
        re.compile(r"\b(정의)\b"),
        re.compile(r"라고\s*(한다|부른다|정의한다)"),
        re.compile(r"\b([^\s]+)\s*(이란|란)\b"),
    ],
    "condition_exception": [
        re.compile(r"\b(만약|하면|하는\s*경우|조건|전제)\b"),
        re.compile(r"\b(단\b|다만|예외|단서)\b"),
        re.compile(r"\b(필요조건|충분조건)\b"),
    ],
    "contrast": [
        re.compile(r"\b(하지만|그러나|반면|그런데)\b"),
        re.compile(r"\b(대신|오히려)\b"),
        re.compile(r"\b(아니다|반대로)\b"),
    ],
    "justification": [
        re.compile(r"\b(왜냐하면|이유는|근거는)\b"),
        re.compile(r"\b(따라서|그러므로|그래서|그러니)\b"),
    ],
    "causal": [
        re.compile(r"\b(원인|때문에|기인)\b"),
        re.compile(r"\b(결과|영향|초래)\b"),
    ],
    "example": [
        re.compile(r"\b(예를\s*들어|가령|예컨대|사례)\b"),
    ],
}

# (B) Bucket compatibility (soft constraints)
# You can tune these; they are deliberately minimal and interpretable.
# Buckets are whatever you used: A/B/C/D/E (or others). We'll assume at least C = boundary/contrast.
_BUCKET_BIAS = {
    # If either endpoint is a boundary/contrast node, contrast edges are more plausible.
    "C": {"contrast": 1.35, "condition_exception": 1.10},
    # If either endpoint is a meta node, definition edges are more plausible.
    "E": {"definition": 1.25, "justification": 1.10},
    # If endpoint is anchor/core-ish (A/D), definition/causal are slightly more plausible.
    "A": {"definition": 1.10, "causal": 1.05},
    "D": {"definition": 1.10, "causal": 1.05},
}

# (C) Transition priors: what tends to follow what around a node in outgoing/incoming edges.
# Example: definitions often connect to exceptions/conditions; contrasts often surround boundary nodes.
_TRANSITION_PRIOR = {
    "definition": {"condition_exception": 1.15, "justification": 1.10},
    "justification": {"causal": 1.10},
    "contrast": {"justification": 1.05, "condition_exception": 1.05},
    "condition_exception": {"justification": 1.05},
    "causal": {"justification": 1.05},
    "example": {"definition": 1.05, "justification": 1.05},
}


def _node_bucket_map(graph_obj: Dict[str, Any]) -> Dict[int, str]:
    m: Dict[int, str] = {}
    for nd in graph_obj.get("nodes", []):
        try:
            m[int(nd["id"])] = str(nd.get("bucket", "Unknown"))
        except Exception:
            continue
    return m


def _hard_rule_vote_from_evidence(edge: Dict[str, Any]) -> Tuple[Optional[str], Dict[str, Any]]:
    """
    Look at evidence_stage2 if present else evidence (stage1),
    apply hard marker counts. If a type dominates enough, force.
    """
    evid = edge.get("evidence_stage2") or edge.get("evidence") or []
    if not isinstance(evid, list) or not evid:
        return None, {"reason": "no_evidence"}

    counts: Dict[str, int] = {t: 0 for t in _CONFIDENT_TYPES}
    for ev in evid:
        s = str(ev.get("sentence", ""))
        for t, pats in _HARD_MARKERS.items():
            if any(p.search(s) for p in pats):
                counts[t] += 1

    best_t = max(counts.items(), key=lambda kv: kv[1])[0]
    best_c = counts[best_t]
    total_hits = sum(counts.values())

    # Force only when we have strong and specific evidence.
    # - at least 2 hits for best type
    # - and best is >= 60% of all hard hits (if any)
    if best_c >= 2 and (total_hits == 0 or (best_c / max(total_hits, 1)) >= 0.60):
        return best_t, {
            "reason": "hard_marker_force",
            "counts": counts,
            "best": best_t,
            "best_hits": best_c,
            "total_hits": total_hits,
        }

    return None, {"reason": "hard_marker_insufficient", "counts": counts, "total_hits": total_hits}


def _bucket_multiplier(bucket_i: str, bucket_j: str, edge_type: str) -> float:
    """
    Multiply score by bucket biases for both endpoints (if any).
    """
    mul = 1.0
    for b in (bucket_i, bucket_j):
        if b in _BUCKET_BIAS and edge_type in _BUCKET_BIAS[b]:
            mul *= float(_BUCKET_BIAS[b][edge_type])
    return mul


def _transition_multiplier(
    local_types_out_i: List[str],
    local_types_in_j: List[str],
    candidate: str,
) -> float:
    """
    If neighborhood around endpoints suggests candidate (via prior),
    slightly boost it.
    """
    mul = 1.0

    # From outgoing types of i: if many are of type T, candidate may follow T via transition prior.
    for t in local_types_out_i:
        mul *= float(_TRANSITION_PRIOR.get(t, {}).get(candidate, 1.0))

    # From incoming types of j: if many are of type T, candidate may precede T; we reuse same priors symmetrically.
    for t in local_types_in_j:
        mul *= float(_TRANSITION_PRIOR.get(t, {}).get(candidate, 1.0))

    # Prevent explosion
    return float(min(2.0, max(0.5, mul)))


def _collect_local_types(
    out_adj: Dict[int, List[Tuple[int, Dict[str, Any]]]],
    in_adj: Dict[int, List[Tuple[int, Dict[str, Any]]]],
    i: int,
    j: int,
    *,
    take_top: int = 8,
) -> Tuple[List[str], List[str]]:
    """
    Returns (top outgoing edge types from i, top incoming edge types to j),
    using weight*confidence ordering.
    """
    def rank(edges: List[Tuple[int, Dict[str, Any]]]) -> List[Dict[str, Any]]:
        ranked = []
        for _, e in edges:
            t = _type_of_edge(e)
            if t not in _CONFIDENT_TYPES:
                continue
            w = _edge_weight(e)
            c = _edge_confidence_from_decision(_get_best_decision_payload(e))
            ranked.append((w * c, e))
        ranked.sort(key=lambda x: x[0], reverse=True)
        return [e for _, e in ranked[:take_top]]

    outs = rank(out_adj.get(i, []))
    ins = rank(in_adj.get(j, []))

    out_types = [_type_of_edge(e) for e in outs]
    in_types = [_type_of_edge(e) for e in ins]
    return out_types, in_types


def _stage3_scores_if_present(edge: Dict[str, Any]) -> Optional[Dict[str, float]]:
    """
    If stage3 decision stored scores, reuse them. Otherwise None.
    """
    d3 = edge.get("edge_type_decision_stage3")
    if isinstance(d3, dict):
        sc = d3.get("scores")
        if isinstance(sc, dict) and all(k in _CONFIDENT_TYPES for k in sc.keys()):
            # ensure floats
            return {k: float(v) for k, v in sc.items()}
    return None


def fourth_stage_edge_type_adjudication_constraint_based(
    graph_obj: Dict[str, Any],
    *,
    target_edge_types: Tuple[str, ...] = ("mixed", "unsure"),
    # decision thresholds
    dominance_ratio: float = 0.66,
    margin_ratio: float = 0.20,
    min_total_score: float = 1e-6,
    priority: Optional[List[str]] = None,
    # behavior
    overwrite_when_confident: bool = True,
    keep_edge_type_when_forced: bool = True,
) -> Dict[str, Any]:
    """
    Stage-4:
      - First apply HARD RULES on evidence -> force type when clear.
      - Else compute candidate scores by combining:
          base graph score (stage3 scores if present else uniform)
          * bucket multipliers
          * transition multipliers (local neighborhood types)
      - Then decide with dominance + margin thresholds.

    Adds per-edge:
      - edge_type_stage4
      - edge_type_decision_stage4
      - overwrites edge_type when confident (optional)
    """
    priority = priority or _DEFAULT_PRIORITY
    bucket_map = _node_bucket_map(graph_obj)
    out_adj, in_adj = _build_adjacency(graph_obj)

    out = {**graph_obj, "edges": []}
    meta = dict(out.get("meta", {}))
    meta["edge_type_stage4"] = {
        "target_edge_types": list(target_edge_types),
        "dominance_ratio": float(dominance_ratio),
        "margin_ratio": float(margin_ratio),
        "min_total_score": float(min_total_score),
        "priority": priority,
        "overwrite_when_confident": bool(overwrite_when_confident),
        "keep_edge_type_when_forced": bool(keep_edge_type_when_forced),
        "bucket_bias": _BUCKET_BIAS,
        "transition_prior": _TRANSITION_PRIOR,
    }
    out["meta"] = meta

    def decide_from_scores(scores: Dict[str, float]) -> Tuple[str, Dict[str, Any]]:
        total = float(sum(scores.values()))
        if total < min_total_score:
            return "unsure", {"reason": "insufficient_stage4_signal", "total_score": total, "scores": scores}

        items = sorted(scores.items(), key=lambda kv: kv[1], reverse=True)
        best_label, best = items[0]
        second = items[1][1] if len(items) > 1 else 0.0

        # tie-break
        tied = [lab for lab, sc in items if abs(sc - best) <= 1e-12]
        if len(tied) > 1:
            for p in priority:
                if p in tied:
                    best_label = p
                    break

        dom = best / total if total > 0 else 0.0
        margin = (best - second) / max(best, 1e-12)

        if dom >= dominance_ratio and margin >= margin_ratio:
            return best_label, {
                "reason": "stage4_confident",
                "total_score": total,
                "best": best,
                "second": second,
                "dominance_ratio": dom,
                "margin_ratio": margin,
                "thresholds": {"dominance_ratio": dominance_ratio, "margin_ratio": margin_ratio},
                "scores": scores,
            }

        return "mixed", {
            "reason": "stage4_ambiguous",
            "total_score": total,
            "best_label": best_label,
            "best": best,
            "second": second,
            "dominance_ratio": dom,
            "margin_ratio": margin,
            "thresholds": {"dominance_ratio": dominance_ratio, "margin_ratio": margin_ratio},
            "scores": scores,
        }

    for e in graph_obj.get("edges", []):
        e2 = dict(e)
        et = str(e.get("edge_type", "unsure"))

        if et not in target_edge_types:
            out["edges"].append(e2)
            continue

        i = int(e["source"])
        j = int(e["target"])
        bi = bucket_map.get(i, "Unknown")
        bj = bucket_map.get(j, "Unknown")

        # (A) hard rule force
        forced_type, forced_info = _hard_rule_vote_from_evidence(e)
        if forced_type is not None and forced_type in _CONFIDENT_TYPES:
            e2["edge_type_stage4"] = forced_type
            e2["edge_type_decision_stage4"] = forced_info

            if keep_edge_type_when_forced:
                e2["edge_type_prev"] = e2.get("edge_type_prev", e.get("edge_type", None))
                e2["edge_type"] = forced_type
                e2["edge_type_decision"] = forced_info

            out["edges"].append(e2)
            continue

        # (B)(C)(D) score combining
        base_scores = _stage3_scores_if_present(e)
        if base_scores is None:
            # fallback: uniform prior (very conservative)
            base_scores = {t: 1.0 for t in _CONFIDENT_TYPES}

        # local types
        out_types, in_types = _collect_local_types(out_adj, in_adj, i, j)

        scores4: Dict[str, float] = {}
        for t in _CONFIDENT_TYPES:
            s = float(base_scores.get(t, 0.0))
            s *= _bucket_multiplier(bi, bj, t)
            s *= _transition_multiplier(out_types, in_types, t)
            scores4[t] = s

        edge_type4, info4 = decide_from_scores(scores4)
        info4 = {**info4, "forced_info": forced_info, "buckets": {"i": bi, "j": bj}, "local_types": {"out_i": out_types, "in_j": in_types}}

        e2["edge_type_stage4"] = edge_type4
        e2["edge_type_decision_stage4"] = info4

        if overwrite_when_confident and edge_type4 in _CONFIDENT_TYPES:
            e2["edge_type_prev"] = e2.get("edge_type_prev", e.get("edge_type", None))
            e2["edge_type"] = edge_type4
            e2["edge_type_decision"] = info4

        out["edges"].append(e2)

    return out
```

### 4차 판정기의 “결정 원리”를 한 문장으로

**증거(문장 마커)로 강제할 수 있으면 강제하고, 아니면 3차(그래프 점수)에 “버킷 적합성 + 전이 사전확률”을 곱해서 가장 그럴듯한 타입만 남깁니다.**

### 바로 튜닝 가능한 레버 3개

- `dominance_ratio_stage4`, `margin_ratio_stage4`: 확정 기준을 빡세게/느슨하게
    
- `_BUCKET_BIAS`: C/E 버킷에서 어떤 타입을 더 믿을지
    
- `_TRANSITION_PRIOR`: 정의→예외/근거 같은 흐름을 얼마나 강하게 믿을지
    

원하시면 다음으로, stage4에서도 애매하게 남는 엣지를 대상으로 **“엣지 타입별로 evidence 문장을 다시 검색(마커 포함 문장만)”** 하는 _타입별 리트리버_까지 붙이면, 사실상 `mixed/unsure`가 거의 사라집니다.

```python
from __future__ import annotations

from typing import Dict, List, Sequence, Tuple, Any, Optional
import numpy as np
import re

# Assumes existing:
# - _CONFIDENT_TYPES
# - _HARD_MARKERS (stage4)
# - detect_relation_hints, pick_primary_relation (optional, we can add)
# - promote_edge_type_from_evidence (stage1/2)
# - art.upper.a(x) -> softmax over nodes
# - embed.encode(sentence) -> vector
#
# Optional but recommended:
# - _edge_confidence_from_decision, _get_best_decision_payload (not required here)


# -------------------------
# Type-filtered retriever: restrict evidence pool by marker patterns
# -------------------------

def _collect_sentences(corpus_docs: Sequence[Sequence[str]]) -> List[str]:
    return [s for doc in corpus_docs for s in doc]


def build_type_filtered_sentence_pools(
    corpus_docs: Sequence[Sequence[str]],
    *,
    restrict_to_concept: Optional[str] = None,
    per_type_pool_cap: int = 20000,
    seed: int = 7,
) -> Dict[str, List[str]]:
    """
    Build per-type pools of sentences that match hard marker patterns.
    This is the "type-specific retriever" core.

    Returns:
      pools[type] = list of sentences (possibly capped)
      pools["_all"] = all sentences (for fallback)
    """
    rng = np.random.default_rng(seed)
    all_sents = _collect_sentences(corpus_docs)
    if restrict_to_concept is not None:
        all_sents = [s for s in all_sents if restrict_to_concept in s]

    pools: Dict[str, List[str]] = {"_all": all_sents}

    for t in _CONFIDENT_TYPES:
        pats = _HARD_MARKERS.get(t, [])
        if not pats:
            pools[t] = []
            continue
        matched = [s for s in all_sents if any(p.search(s) for p in pats)]
        # cap
        if len(matched) > per_type_pool_cap:
            idx = rng.choice(len(matched), size=per_type_pool_cap, replace=False)
            matched = [matched[i] for i in idx]
        pools[t] = matched

    return pools


def _compute_As_for_sentences(
    art,
    embed,
    sentences: List[str],
) -> Tuple[np.ndarray, np.ndarray]:
    """
    Returns:
      Xs: (M,d)
      As: (M,N)
    """
    Xs = np.stack([embed.encode(s) for s in sentences], axis=0).astype(float)
    As = np.stack([art.upper.a(x) for x in Xs], axis=0).astype(float)
    return Xs, As


def retrieve_typed_evidence_for_edge(
    art,
    embed,
    *,
    edge_i: int,
    edge_j: int,
    pools: Dict[str, List[str]],
    candidate_types: Optional[List[str]] = None,
    evidence_per_type: int = 6,
    relation_boost: float = 1.10,
    seed: int = 7,
) -> Dict[str, Any]:
    """
    For a single edge (i->j):
      - for each candidate type T, restrict pool to sentences matching markers of T
      - compute score = a_i * a_j * (1 + boost * has_any_hint)
      - pick top evidence_per_type sentences
      - produce per-type histograms and "best type" suggestion

    Returns:
      dict with typed_evidence[type], typed_stats[type], combined_vote, suggested_type
    """
    rng = np.random.default_rng(seed)
    candidate_types = candidate_types or list(_CONFIDENT_TYPES)

    typed_evidence: Dict[str, List[Dict[str, Any]]] = {}
    typed_stats: Dict[str, Dict[str, Any]] = {}
    vote_hist: Dict[str, int] = {}

    for t in candidate_types:
        sents = pools.get(t, [])
        if not sents:
            typed_evidence[t] = []
            typed_stats[t] = {"pool_size": 0, "note": "empty_pool"}
            continue

        # precompute hints cheaply (optional)
        hint_lists = [detect_relation_hints(s) for s in sents]
        has_any = np.array([1.0 if h else 0.0 for h in hint_lists], dtype=float)

        # compute activations
        _, As = _compute_As_for_sentences(art, embed, sents)
        base = As[:, edge_i] * As[:, edge_j]
        score = base * (1.0 + (relation_boost - 1.0) * has_any)

        topk = np.argsort(score)[::-1][:evidence_per_type]
        evid_list: List[Dict[str, Any]] = []
        primary_hist: Dict[str, int] = {}

        for idx in topk:
            idx = int(idx)
            rels = hint_lists[idx]
            primary = pick_primary_relation(rels)
            if primary is not None:
                primary_hist[primary] = primary_hist.get(primary, 0) + 1

            evid_list.append({
                "sentence": sents[idx],
                "score": float(score[idx]),
                "base_score": float(base[idx]),
                "a_i": float(As[idx, edge_i]),
                "a_j": float(As[idx, edge_j]),
                "relation_hints": rels,
                "primary_relation_hint": primary,
                "retrieved_as_type": t,
            })

        typed_evidence[t] = evid_list
        typed_stats[t] = {
            "pool_size": int(len(sents)),
            "primary_relation_hist": primary_hist,
            "top_score": float(score[int(topk[0])]) if len(topk) else 0.0,
            "mean_top_score": float(np.mean([ev["score"] for ev in evid_list])) if evid_list else 0.0,
        }

        # vote for the retrieval type itself if it produced reasonable evidence
        if evid_list:
            vote_hist[t] = vote_hist.get(t, 0) + len(evid_list)

    # Suggested type: highest mean_top_score among types with evidence
    best_type = None
    best_score = -1.0
    for t, st in typed_stats.items():
        sc = float(st.get("mean_top_score", 0.0) or 0.0)
        if sc > best_score and typed_evidence.get(t):
            best_score = sc
            best_type = t

    return {
        "edge": {"i": int(edge_i), "j": int(edge_j)},
        "typed_evidence": typed_evidence,
        "typed_stats": typed_stats,
        "suggested_type_by_score": best_type,
        "suggested_type_score": float(best_score),
        "vote_hist": vote_hist,
    }


def apply_type_retriever_to_ambiguous_edges(
    art,
    corpus_docs: Sequence[Sequence[str]],
    embed,
    graph_obj: Dict[str, Any],
    *,
    target_edge_types: Tuple[str, ...] = ("mixed", "unsure"),
    restrict_evidence_to_concept_contexts: bool = False,
    per_type_pool_cap: int = 20000,
    evidence_per_type: int = 6,
    relation_boost: float = 1.10,
    # final decision rules after typed retrieval
    min_types_with_evidence: int = 1,
    score_dominance_ratio: float = 1.15,
    overwrite_when_confident: bool = True,
    seed: int = 7,
) -> Dict[str, Any]:
    """
    For each ambiguous edge:
      - build per-type sentence pools once
      - retrieve typed evidence for each candidate type
      - decide:
          if best_type exists and its mean_top_score >= score_dominance_ratio * second_best_mean
          => adopt best_type
        else keep ambiguous but attach typed evidence payload for inspection.

    Adds:
      edge["edge_type_stage5_retriever"]
      edge["edge_type_decision_stage5_retriever"]
      edge["typed_evidence"] (heavy)  # you can disable if too big
    """
    concept = art.concept if restrict_evidence_to_concept_contexts else None
    pools = build_type_filtered_sentence_pools(
        corpus_docs,
        restrict_to_concept=concept,
        per_type_pool_cap=per_type_pool_cap,
        seed=seed,
    )

    out = {**graph_obj, "edges": []}
    meta = dict(out.get("meta", {}))
    meta["edge_type_stage5_retriever"] = {
        "target_edge_types": list(target_edge_types),
        "restrict_evidence_to_concept_contexts": bool(restrict_evidence_to_concept_contexts),
        "per_type_pool_cap": int(per_type_pool_cap),
        "evidence_per_type": int(evidence_per_type),
        "relation_boost": float(relation_boost),
        "score_dominance_ratio": float(score_dominance_ratio),
        "overwrite_when_confident": bool(overwrite_when_confident),
    }
    out["meta"] = meta

    for e in graph_obj.get("edges", []):
        e2 = dict(e)
        et = str(e.get("edge_type", "unsure"))

        if et not in target_edge_types:
            out["edges"].append(e2)
            continue

        i = int(e["source"])
        j = int(e["target"])

        payload = retrieve_typed_evidence_for_edge(
            art,
            embed,
            edge_i=i,
            edge_j=j,
            pools=pools,
            evidence_per_type=evidence_per_type,
            relation_boost=relation_boost,
            seed=seed,
        )

        # Decide by mean_top_score dominance
        # collect mean_top_score per type
        mean_scores: List[Tuple[str, float]] = []
        for t, st in payload["typed_stats"].items():
            ms = float(st.get("mean_top_score", 0.0) or 0.0)
            if payload["typed_evidence"].get(t):
                mean_scores.append((t, ms))
        mean_scores.sort(key=lambda kv: kv[1], reverse=True)

        decision = {
            "reason": "typed_retrieval_insufficient",
            "mean_top_scores": mean_scores,
            "suggested_type_by_score": payload["suggested_type_by_score"],
            "suggested_type_score": payload["suggested_type_score"],
            "thresholds": {"score_dominance_ratio": score_dominance_ratio},
        }

        chosen = None
        if len(mean_scores) >= min_types_with_evidence and mean_scores:
            best_t, best_ms = mean_scores[0]
            second_ms = mean_scores[1][1] if len(mean_scores) > 1 else 0.0
            # dominance check
            if second_ms <= 1e-12:
                # only one type had evidence
                chosen = best_t
                decision["reason"] = "typed_retrieval_single_type"
            elif best_ms >= score_dominance_ratio * second_ms:
                chosen = best_t
                decision["reason"] = "typed_retrieval_dominant"
                decision["dominance_ratio_observed"] = float(best_ms / max(second_ms, 1e-12))
            else:
                decision["reason"] = "typed_retrieval_ambiguous"
                decision["dominance_ratio_observed"] = float(best_ms / max(second_ms, 1e-12))

        e2["edge_type_stage5_retriever"] = chosen or et
        e2["edge_type_decision_stage5_retriever"] = decision
        e2["typed_evidence"] = payload["typed_evidence"]
        e2["typed_stats"] = payload["typed_stats"]

        if overwrite_when_confident and chosen is not None:
            e2["edge_type_prev"] = e2.get("edge_type_prev", e.get("edge_type", None))
            e2["edge_type"] = chosen
            e2["edge_type_decision"] = decision

        out["edges"].append(e2)

    return out
```

### 이 “타입별 리트리버”가 실제로 하는 일

- **타입마다**(정의/예외/대립/근거/인과/예시) 그 타입을 강하게 암시하는 **문장 마커**가 있는 문장만 따로 모아 “풀(pool)”을 만듭니다.
    
- 애매한 엣지(i→j)에 대해,
    
    - 각 타입 풀에서만 evidence를 다시 뽑고,
        
    - `a_i * a_j`가 큰 문장들을 타입별로 비교해서
        
    - **어느 타입 풀에서 가장 ‘좋은 증거’가 많이/강하게 나오느냐**로 최종 결정을 시도합니다.
        
- 결과는 `typed_evidence`, `typed_stats`로 그대로 남아서, 나중에 사람이 보면서 규칙 수정하기도 쉽습니다.
    

원하시면 다음으로는 `typed_evidence`가 JSON을 너무 비대하게 만들 수 있으니,

- “본문은 저장하지 않고 문장 해시+인덱스만 저장” 모드,
    
- 또는 “상위 2개 타입만 저장” 모드,  
    를 붙여서 파일 크기 제어까지 같이 해드릴게요.

```python
# File-size control helpers for stage5 typed retriever payloads
# -------------------------

def _stable_sentence_id(sentence: str, *, algo: str = "sha1") -> str:
    """
    Stable short id for a sentence, for JSON slimming.
    """
    h = hashlib.new(algo)
    h.update(sentence.encode("utf-8", errors="ignore"))
    return h.hexdigest()[:16]


def slim_typed_evidence_payload(
    edge: Dict[str, Any],
    *,
    mode: str = "hash_only",   # "full" | "hash_only" | "top2_full" | "top2_hash"
    keep_top_k_types: int = 2, # used by top2_* modes
    keep_top_m_per_type: int = 3,  # cap per type
    keep_fields: Optional[List[str]] = None,
) -> Dict[str, Any]:
    """
    Reduce JSON size of stage5 typed retriever payloads.

    Expects edge has:
      - edge["typed_evidence"] : {type: [ {sentence, score, ...}, ...]}
      - edge["typed_stats"]    : {type: {...}}

    Modes:
      - "full": keep typed_evidence but cap per type to keep_top_m_per_type
      - "hash_only": remove full sentences; store sentence_id + a few numeric fields
      - "top2_full": keep only top K types (by typed_stats.mean_top_score) with capped evidence
      - "top2_hash": keep only top K types, store sentence_id only

    keep_fields (for hash_only/top2_hash):
      which fields to retain per evidence record besides sentence_id.
      default keeps minimal useful debug.
    """
    keep_fields = keep_fields or ["score", "base_score", "a_i", "a_j", "primary_relation_hint", "retrieved_as_type"]

    typed_evidence = edge.get("typed_evidence")
    typed_stats = edge.get("typed_stats")

    if not isinstance(typed_evidence, dict) or not isinstance(typed_stats, dict):
        return edge  # nothing to slim

    # Determine type ranking by mean_top_score
    mean_scores: List[Tuple[str, float]] = []
    for t, st in typed_stats.items():
        ms = float((st or {}).get("mean_top_score", 0.0) or 0.0)
        # only count if evidence exists
        if isinstance(typed_evidence.get(t), list) and typed_evidence.get(t):
            mean_scores.append((t, ms))
    mean_scores.sort(key=lambda kv: kv[1], reverse=True)

    top_types = [t for t, _ in mean_scores[:keep_top_k_types]]

    def cap_list(lst: List[Dict[str, Any]]) -> List[Dict[str, Any]]:
        return lst[:keep_top_m_per_type]

    new_typed_evidence: Dict[str, Any] = {}
    new_typed_stats: Dict[str, Any] = {}

    def convert_record_hash(rec: Dict[str, Any]) -> Dict[str, Any]:
        s = str(rec.get("sentence", ""))
        out = {"sentence_id": _stable_sentence_id(s)}
        for k in keep_fields:
            if k in rec:
                out[k] = rec[k]
        return out

    if mode == "full":
        # keep all types, but cap evidence
        for t, lst in typed_evidence.items():
            if isinstance(lst, list):
                new_typed_evidence[t] = cap_list(lst)
            else:
                new_typed_evidence[t] = lst
        new_typed_stats = typed_stats

    elif mode == "hash_only":
        for t, lst in typed_evidence.items():
            if isinstance(lst, list):
                new_typed_evidence[t] = [convert_record_hash(r) for r in cap_list(lst)]
            else:
                new_typed_evidence[t] = lst
        new_typed_stats = typed_stats

    elif mode == "top2_full":
        for t in top_types:
            lst = typed_evidence.get(t, [])
            if isinstance(lst, list):
                new_typed_evidence[t] = cap_list(lst)
            else:
                new_typed_evidence[t] = lst
            if t in typed_stats:
                new_typed_stats[t] = typed_stats[t]
        # keep minimal ranking for transparency
        new_typed_stats["_ranking"] = mean_scores[: max(keep_top_k_types, 2)]

    elif mode == "top2_hash":
        for t in top_types:
            lst = typed_evidence.get(t, [])
            if isinstance(lst, list):
                new_typed_evidence[t] = [convert_record_hash(r) for r in cap_list(lst)]
            else:
                new_typed_evidence[t] = lst
            if t in typed_stats:
                new_typed_stats[t] = typed_stats[t]
        new_typed_stats["_ranking"] = mean_scores[: max(keep_top_k_types, 2)]

    else:
        raise ValueError(f"Unknown slimming mode: {mode}")

    e2 = dict(edge)
    e2["typed_evidence"] = new_typed_evidence
    e2["typed_stats"] = new_typed_stats
    e2["typed_evidence_slim_mode"] = mode
    e2["typed_evidence_slim_config"] = {
        "keep_top_k_types": int(keep_top_k_types),
        "keep_top_m_per_type": int(keep_top_m_per_type),
        "keep_fields": keep_fields,
    }
    return e2


def slim_graph_stage5_payload(
    graph_obj: Dict[str, Any],
    *,
    mode: str = "top2_hash",
    keep_top_k_types: int = 2,
    keep_top_m_per_type: int = 3,
    keep_fields: Optional[List[str]] = None,
    only_if_stage5_present: bool = True,
) -> Dict[str, Any]:
    """
    Apply payload slimming to all edges in graph_obj.

    If only_if_stage5_present is True:
      only edges with "typed_evidence" key are processed.
    """
    out = {**graph_obj, "edges": []}
    for e in graph_obj.get("edges", []):
        if only_if_stage5_present and "typed_evidence" not in e:
            out["edges"].append(e)
            continue
        out["edges"].append(
            slim_typed_evidence_payload(
                e,
                mode=mode,
                keep_top_k_types=keep_top_k_types,
                keep_top_m_per_type=keep_top_m_per_type,
                keep_fields=keep_fields,
            )
        )

    meta = dict(out.get("meta", {}))
    meta["typed_evidence_slimming"] = {
        "mode": mode,
        "keep_top_k_types": int(keep_top_k_types),
        "keep_top_m_per_type": int(keep_top_m_per_type),
        "keep_fields": keep_fields or ["score", "base_score", "a_i", "a_j", "primary_relation_hint", "retrieved_as_type"],
        "only_if_stage5_present": bool(only_if_stage5_present),
    }
    out["meta"] = meta
    return out

# or storing a global sentence_id -> sentence table separately.
```

### 실무적으로 추천하는 기본값

- `mode="top2_hash"`, `keep_top_m_per_type=3`
    
    - 엣지마다 “가장 유력한 2개 타입”만 남기고,
        
    - 각 타입에서 상위 3문장만,
        
    - 문장 본문은 해시(`sentence_id`)로 대체합니다.
        

원하시면 다음 단계로, **전역 테이블**(한 번만 저장): `sentence_id -> sentence` 를 별도 파일로 분리해서, 그래프 JSON은 가볍게 유지하면서도 필요할 때만 본문을 복원하는 구조까지 같이 붙여드릴 수 있습니다.

```python
from __future__ import annotations

from typing import Dict, List, Sequence, Tuple, Any, Optional
import hashlib
import json


# -------------------------
# Global sentence table (sentence_id -> sentence) + rehydration utilities
# -------------------------

def sentence_id(sentence: str, *, algo: str = "sha1", nhex: int = 16) -> str:
    h = hashlib.new(algo)
    h.update(sentence.encode("utf-8", errors="ignore"))
    return h.hexdigest()[:nhex]


def build_sentence_table_from_corpus(
    corpus_docs: Sequence[Sequence[str]],
    *,
    algo: str = "sha1",
    nhex: int = 16,
    dedupe: bool = True,
) -> Dict[str, str]:
    """
    Build a global mapping: sentence_id -> sentence.

    If dedupe=True, duplicate sentences collapse automatically (same hash).
    Collision risk exists but is extremely low for sha1/16hex for typical corpora;
    if you want safer, set nhex=24 or 32.
    """
    table: Dict[str, str] = {}
    for doc in corpus_docs:
        for s in doc:
            sid = sentence_id(s, algo=algo, nhex=nhex)
            if dedupe:
                table.setdefault(sid, s)
            else:
                # keep last occurrence (not recommended)
                table[sid] = s
    return table


def attach_sentence_table_reference(
    graph_obj: Dict[str, Any],
    *,
    table_name: str = "sentence_table.json",
    algo: str = "sha1",
    nhex: int = 16,
) -> Dict[str, Any]:
    """
    Store reference metadata in graph JSON (does NOT embed the table itself).
    """
    out = dict(graph_obj)
    meta = dict(out.get("meta", {}))
    meta["sentence_table_ref"] = {
        "filename": table_name,
        "algo": algo,
        "nhex": int(nhex),
        "key": "sentence_id",
    }
    out["meta"] = meta
    return out


def save_json(path: str, obj: Any) -> None:
    with open(path, "w", encoding="utf-8") as f:
        json.dump(obj, f, ensure_ascii=False, indent=2)


def load_json(path: str) -> Any:
    with open(path, "r", encoding="utf-8") as f:
        return json.load(f)


# -------------------------
# Rehydration: replace sentence_id with actual sentence text
# -------------------------

def rehydrate_typed_evidence_in_edge(
    edge: Dict[str, Any],
    sentence_table: Dict[str, str],
    *,
    in_place: bool = False,
    field: str = "typed_evidence",
) -> Dict[str, Any]:
    """
    Rehydrate edge[field] records that have "sentence_id" into "sentence".
    Keeps sentence_id as well.
    """
    e = edge if in_place else dict(edge)

    te = e.get(field)
    if not isinstance(te, dict):
        return e

    te2: Dict[str, Any] = {}
    for t, lst in te.items():
        if not isinstance(lst, list):
            te2[t] = lst
            continue
        new_list = []
        for rec in lst:
            if not isinstance(rec, dict):
                new_list.append(rec)
                continue
            rec2 = dict(rec)
            sid = rec2.get("sentence_id")
            if isinstance(sid, str) and "sentence" not in rec2:
                rec2["sentence"] = sentence_table.get(sid)
            new_list.append(rec2)
        te2[t] = new_list

    e[field] = te2
    return e


def rehydrate_graph_typed_evidence(
    graph_obj: Dict[str, Any],
    sentence_table: Dict[str, str],
    *,
    in_place: bool = False,
    field: str = "typed_evidence",
) -> Dict[str, Any]:
    """
    Rehydrate all edges in a graph JSON.
    """
    g = graph_obj if in_place else {**graph_obj, "edges": []}
    edges = graph_obj.get("edges", [])
    if in_place:
        for idx, e in enumerate(edges):
            edges[idx] = rehydrate_typed_evidence_in_edge(e, sentence_table, in_place=False, field=field)
        return graph_obj

    for e in edges:
        g["edges"].append(rehydrate_typed_evidence_in_edge(e, sentence_table, in_place=False, field=field))
    return g


# -------------------------
# End-to-end: run stage5, slim, split into two files
# -------------------------

def export_with_split_sentence_table(
    art,
    corpus_docs,
    embed,
    *,
    # run full pipeline with stage5 retriever (you already have this)
    graph_export_fn,
    graph_export_kwargs: Optional[Dict[str, Any]] = None,

    # slimming params
    slim_mode: str = "top2_hash",
    keep_top_k_types: int = 2,
    keep_top_m_per_type: int = 3,
    keep_fields: Optional[List[str]] = None,

    # sentence table params
    table_algo: str = "sha1",
    table_nhex: int = 16,
    table_filename: str = "sentence_table.json",

    # output file names
    graph_filename: str = "concept_graph_slim.json",
) -> Tuple[Dict[str, Any], Dict[str, str]]:
    """
    Produces:
      - slim graph JSON referencing an external sentence table
      - sentence_table JSON mapping sentence_id->sentence
    Returns in-memory objects; you can call save_json on them.
    """
    graph_export_kwargs = graph_export_kwargs or {}
    g = graph_export_fn(art, corpus_docs, embed, **graph_export_kwargs)

    # slim the heavy payloads
    g_slim = slim_graph_stage5_payload(
        g,
        mode=slim_mode,
        keep_top_k_types=keep_top_k_types,
        keep_top_m_per_type=keep_top_m_per_type,
        keep_fields=keep_fields,
        only_if_stage5_present=True,
    )

    # build global table
    table = build_sentence_table_from_corpus(
        corpus_docs,
        algo=table_algo,
        nhex=table_nhex,
        dedupe=True,
    )

    # attach reference meta (without embedding the table)
    g_slim = attach_sentence_table_reference(
        g_slim,
        table_name=table_filename,
        algo=table_algo,
        nhex=table_nhex,
    )

    # recommended: also record the graph file name in meta (optional)
    meta = dict(g_slim.get("meta", {}))
    meta["files"] = {"graph": graph_filename, "sentence_table": table_filename}
    g_slim["meta"] = meta

    return g_slim, table
```
