# 남은 작업
남은 작업은 “돌아가게 만들기” 관점에서 보면 딱 8단계입니다. (지금까지는 판정기/리트리버는 거의 완성이고, **연결·검증·운영**이 남았습니다.)

## 1) 전체 파이프라인 합치기

- 목표: `run(concept, corpus) -> graph_slim.json + sentence_table.json` 한 방에 나오게.
    
- 해야 할 것
    
    - stage1~5 호출 순서 고정
        
    - 각 stage가 기대하는 키(`edge_type_decision`, `evidence_stage2`, `bucket` 등) 존재 여부 점검
        
    - `export_with_split_sentence_table()`에 실제 `graph_export_fn` 연결
        

## 2) “노드 사전(개념 노드)” 실제 생성기 확정

- 목표: 개념 노드가 재현가능하게 생성되고, ID가 안정적으로 고정되게.
    
- 해야 할 것
    
    - 개념 후보 추출(형태소/명사구/용어집/사용자 seed 등) → 노드 리스트
        
    - 동일 개념 병합 규칙(표기 변형, 동의어, 띄어쓰기)
        
    - 노드 ID 정책(해시/증분/정규화 문자열 기반)
        

## 3) 문장 분할·정제(코퍼스 전처리) 규격화

- 목표: evidence가 “문장 단위”로 일관되게 나오고, 해시 테이블도 안정적이게.
    
- 해야 할 것
    
    - 문장 분할 규칙(마침표/줄바꿈/인용/괄호 처리)
        
    - 너무 짧은 문장 제거, 너무 긴 문장 자르기(예: 300~500자)
        
    - 중복 문장 처리(완전중복/유사중복)
        

## 4) 임베딩/활성화 함수 인터페이스 고정

- 목표: `embed.encode()`와 `art.upper.a()`가 실제로 붙어서 일관된 수치가 나오게.
    
- 해야 할 것
    
    - 임베딩 차원/정규화(L2) 여부 고정
        
    - `a(x)` 정의(softmax 온도, 상위 k sparsify 여부)
        
    - 성능 병목이면 캐싱(문장 임베딩 캐시, As 캐시)
        

## 5) 엣지 후보 생성(그래프 스켈레톤) 확정

- 목표: “어떤 노드쌍을 엣지 후보로 볼지”를 재현가능하게.
    
- 해야 할 것
    
    - `top_edges` 뽑는 기준 고정(예: 공활성 a_i*a_j 합의 상위)
        
    - self-loop 금지, 대칭 처리(유향/무향) 정책
        
    - edge weight 정의(문장별 max? 평균? 누적합?)
        

## 6) 라벨링 품질 평가(자동 평가 지표 + 샘플링)

- 목표: 판정기가 잘 붙는지 숫자로 확인.
    
- 해야 할 것
    
    - 샘플 50~200개 엣지에 대해 사람이 정답 라벨링(최소한)
        
    - 지표: 정확도/매크로 F1 + `mixed/unsure` 비율
        
    - stage별 ablation: (1) stage2만, (2) stage2+3, (3) +4, (4) +5 비교
        

## 7) 한국어 마커/힌트 룰 튜닝

- 목표: 타입별 리트리버와 hard marker가 실제 코퍼스 문체에 맞게.
    
- 해야 할 것
    
    - `_HARD_MARKERS` 확장/정제(“~인 경우”, “~에 따라”, “~로서” 같은 패턴)
        
    - `pick_primary_relation()` 우선순위 조정
        
    - 충돌 패턴(예: “따라서”가 근거/인과 둘 다 뜨는 경우) 해결
        

## 8) 운영 옵션(파일 크기, 재현성, 디버깅) 마무리

- 목표: 대용량 코퍼스에서도 “돌리고 저장하고 열어볼 수 있게”.
    
- 해야 할 것
    
    - 기본 출력: `concept_graph_slim.json` + `sentence_table.json`
        
    - 디버그 모드: top2_full + 문장 본문 포함 옵션
        
    - 랜덤성 고정: seed를 메타에 기록, 버전 태깅(규칙/파라미터)

```python
def _now_ms() -> int:
    return int(time.time() * 1000)

def _ensure_dir(path: str) -> None:
    os.makedirs(path, exist_ok=True)

def _save_json(path: str, obj: Any) -> None:
    with open(path, "w", encoding="utf-8") as f:
        json.dump(obj, f, ensure_ascii=False, indent=2)

def _short_hash(text: str, nhex: int = 10) -> str:
    h = hashlib.sha1(text.encode("utf-8", errors="ignore")).hexdigest()
    return h[:nhex]


def normalize_corpus_docs(
    corpus: Union[Sequence[str], Sequence[Sequence[str]]],
    *,
    drop_empty: bool = True,
    strip: bool = True,
) -> List[List[str]]:
    """
    Accepts:
      - list[str] (single doc), or
      - list[list[str]] (multi-doc)
    Returns: list[list[str]]
    """
    if not corpus:
        return []

    if isinstance(corpus[0], str):  # type: ignore[index]
        docs = [list(corpus)]  # type: ignore[arg-type]
    else:
        docs = [list(doc) for doc in corpus]  # type: ignore[assignment]

    out: List[List[str]] = []
    for doc in docs:
        cleaned: List[str] = []
        for s in doc:
            if s is None:
                continue
            ss = str(s)
            if strip:
                ss = ss.strip()
            if drop_empty and not ss:
                continue
            cleaned.append(ss)
        out.append(cleaned)
    return out


def summarize_graph(graph_obj: Dict[str, Any]) -> Dict[str, Any]:
    edges = graph_obj.get("edges", []) or []
    nodes = graph_obj.get("nodes", []) or []

    type_hist: Dict[str, int] = {}
    for e in edges:
        t = str(e.get("edge_type", "unsure"))
        type_hist[t] = type_hist.get(t, 0) + 1

    ambig = type_hist.get("mixed", 0) + type_hist.get("unsure", 0)
    total = len(edges)
    ambig_ratio = (ambig / total) if total > 0 else 0.0

    return {
        "n_nodes": len(nodes),
        "n_edges": total,
        "edge_type_hist": dict(sorted(type_hist.items(), key=lambda kv: (-kv[1], kv[0]))),
        "ambiguous_edges": int(ambig),
        "ambiguous_ratio": float(ambig_ratio),
    }

# -------------------------

def run_end_to_end(
    *,
    concept: str,
    corpus: Union[Sequence[str], Sequence[Sequence[str]]],
    embed: Any,
    config: RunnerConfig = RunnerConfig(),
    # optional: if you already built art upstream, pass it
    art: Optional[Any] = None,
) -> Dict[str, Any]:
    """
    Executes:
      1) normalize corpus
      2) build art (unless provided)
      3) export graph with stage2..5
      4) slim stage5 payload
      5) build sentence_table.json
      6) attach reference + save outputs
      7) generate report json

    Returns report dict (also saved).
    """
    t0 = _now_ms()
    _ensure_dir(config.out_dir)

    corpus_docs = normalize_corpus_docs(corpus)
    if not corpus_docs or sum(len(d) for d in corpus_docs) == 0:
        raise ValueError("Empty corpus after normalization.")

    # 1) Build artifacts
    t1 = _now_ms()
    if art is None:
        # You can change args based on your pipeline signature.
        art = run_pipeline_with_core_x0(concept=concept, corpus_docs=corpus_docs, embed=embed, seed=config.seed)
    t2 = _now_ms()

    # 2) Export graph with adjudicators + retriever
    g = export_graph_json_full_with_stage2_stage3_stage4_stage5_retriever(
        art=art,
        corpus_docs=corpus_docs,
        embed=embed,
        top_edges=config.top_edges,
        evidence_per_edge=config.evidence_per_edge,
        evidence_pool_stage1=config.evidence_pool_stage1,
        seed=config.seed,
        relation_boost=1.10,
        restrict_evidence_to_concept_contexts=config.restrict_evidence_to_concept_contexts,
        expanded_evidence_per_edge=config.expanded_evidence_per_edge,
        evidence_pool_stage2=config.evidence_pool_stage2,
        max_k=config.max_k_stage3,
        alpha_triad=config.alpha_triad,
        alpha_nei=config.alpha_nei,
        per_type_pool_cap=config.per_type_pool_cap,
        evidence_per_type=config.evidence_per_type,
        score_dominance_ratio=config.score_dominance_ratio,
        overwrite_when_confident=True,
    )
    t3 = _now_ms()

    # 3) Slim the heavy payloads
    g_slim = slim_graph_stage5_payload(
        g,
        mode=config.slim_mode,
        keep_top_k_types=config.keep_top_k_types,
        keep_top_m_per_type=config.keep_top_m_per_type,
        keep_fields=["score", "base_score", "a_i", "a_j", "primary_relation_hint", "retrieved_as_type"],
        only_if_stage5_present=True,
    )
    t4 = _now_ms()

    # 4) Build global sentence table (for rehydration)
    sent_table = build_sentence_table_from_corpus(
        corpus_docs,
        algo=config.sentence_id_algo,
        nhex=config.sentence_id_nhex,
        dedupe=True,
    )
    t5 = _now_ms()

    # 5) Attach sentence table reference
    g_slim = attach_sentence_table_reference(
        g_slim,
        table_name=config.sentence_table_filename,
        algo=config.sentence_id_algo,
        nhex=config.sentence_id_nhex,
    )

    # 6) Add run meta
    meta = dict(g_slim.get("meta", {}))
    meta["run"] = {
        "concept": concept,
        "pipeline_version": config.pipeline_version,
        "seed": config.seed,
        "restrict_evidence_to_concept_contexts": config.restrict_evidence_to_concept_contexts,
        "corpus_stats": {
            "n_docs": len(corpus_docs),
            "n_sentences": int(sum(len(d) for d in corpus_docs)),
        },
        "files": {
            "graph": config.graph_filename,
            "sentence_table": config.sentence_table_filename,
            "report": config.report_filename,
        },
        "timing_ms": {
            "normalize": int(t1 - t0),
            "build_art": int(t2 - t1),
            "export_graph": int(t3 - t2),
            "slim": int(t4 - t3),
            "sentence_table": int(t5 - t4),
            "total": int(t5 - t0),
        },
    }
    g_slim["meta"] = meta

    # 7) Save outputs
    graph_path = os.path.join(config.out_dir, config.graph_filename)
    table_path = os.path.join(config.out_dir, config.sentence_table_filename)
    report_path = os.path.join(config.out_dir, config.report_filename)

    _save_json(graph_path, g_slim)
    _save_json(table_path, sent_table)

    # 8) Report
    graph_summary = summarize_graph(g_slim)
    report = {
        "concept": concept,
        "out_dir": config.out_dir,
        "files": meta["run"]["files"],
        "graph_summary": graph_summary,
        "timing_ms": meta["run"]["timing_ms"],
        "config": asdict(config),
        "graph_meta_digest": _short_hash(json.dumps(g_slim.get("meta", {}), ensure_ascii=False)),
    }
    _save_json(report_path, report)

    return report
```

## `src/seayu/runner.py`

```python
from __future__ import annotations

from dataclasses import dataclass, asdict
from typing import Any, Dict, List, Optional, Sequence, Union
import json
import os
import time
import hashlib

from seayu.core.pipeline import run_pipeline_with_core_x0
from seayu.graph.export import export_graph_json_full_with_stage2_stage3_stage4_stage5_retriever
from seayu.graph.summarize import summarize_graph

from seayu.io.corpus import normalize_corpus_docs
from seayu.io.jsonio import save_json
from seayu.io.slimming import slim_graph_stage5_payload
from seayu.io.sentence_table import build_sentence_table_from_corpus, attach_sentence_table_reference


def _now_ms() -> int:
    return int(time.time() * 1000)


def _ensure_dir(path: str) -> None:
    os.makedirs(path, exist_ok=True)


def _short_hash(text: str, nhex: int = 10) -> str:
    return hashlib.sha1(text.encode("utf-8", errors="ignore")).hexdigest()[:nhex]


@dataclass
class RunnerConfig:
    out_dir: str = "./out"
    graph_filename: str = "concept_graph_slim.json"
    sentence_table_filename: str = "sentence_table.json"
    report_filename: str = "run_report.json"

    restrict_evidence_to_concept_contexts: bool = False

    # stage knobs
    top_edges: int = 200
    evidence_per_edge: int = 4
    evidence_pool_stage1: int = 4000

    expanded_evidence_per_edge: int = 12
    evidence_pool_stage2: int = 12000

    max_k_stage3: int = 50
    alpha_triad: float = 0.70
    alpha_nei: float = 0.30

    per_type_pool_cap: int = 20000
    evidence_per_type: int = 6
    score_dominance_ratio: float = 1.15

    # slimming
    slim_mode: str = "top2_hash"
    keep_top_k_types: int = 2
    keep_top_m_per_type: int = 3

    # sentence id
    sentence_id_algo: str = "sha1"
    sentence_id_nhex: int = 16

    seed: int = 7
    pipeline_version: str = "v0.1"


def run_end_to_end(
    *,
    concept: str,
    corpus: Union[Sequence[str], Sequence[Sequence[str]]],
    embed: Any,
    config: RunnerConfig = RunnerConfig(),
    art: Optional[Any] = None,
) -> Dict[str, Any]:
    t0 = _now_ms()
    _ensure_dir(config.out_dir)

    corpus_docs = normalize_corpus_docs(corpus)
    if not corpus_docs or sum(len(d) for d in corpus_docs) == 0:
        raise ValueError("Empty corpus after normalization.")

    t1 = _now_ms()
    if art is None:
        art = run_pipeline_with_core_x0(concept=concept, corpus_docs=corpus_docs, embed=embed, seed=config.seed)
    t2 = _now_ms()

    g = export_graph_json_full_with_stage2_stage3_stage4_stage5_retriever(
        art=art,
        corpus_docs=corpus_docs,
        embed=embed,
        top_edges=config.top_edges,
        evidence_per_edge=config.evidence_per_edge,
        evidence_pool_stage1=config.evidence_pool_stage1,
        seed=config.seed,
        relation_boost=1.10,
        restrict_evidence_to_concept_contexts=config.restrict_evidence_to_concept_contexts,
        expanded_evidence_per_edge=config.expanded_evidence_per_edge,
        evidence_pool_stage2=config.evidence_pool_stage2,
        max_k=config.max_k_stage3,
        alpha_triad=config.alpha_triad,
        alpha_nei=config.alpha_nei,
        per_type_pool_cap=config.per_type_pool_cap,
        evidence_per_type=config.evidence_per_type,
        score_dominance_ratio=config.score_dominance_ratio,
        overwrite_when_confident=True,
    )
    t3 = _now_ms()

    g_slim = slim_graph_stage5_payload(
        g,
        mode=config.slim_mode,
        keep_top_k_types=config.keep_top_k_types,
        keep_top_m_per_type=config.keep_top_m_per_type,
        keep_fields=["score", "base_score", "a_i", "a_j", "primary_relation_hint", "retrieved_as_type"],
        only_if_stage5_present=True,
    )
    t4 = _now_ms()

    sent_table = build_sentence_table_from_corpus(
        corpus_docs,
        algo=config.sentence_id_algo,
        nhex=config.sentence_id_nhex,
        dedupe=True,
    )
    t5 = _now_ms()

    g_slim = attach_sentence_table_reference(
        g_slim,
        table_name=config.sentence_table_filename,
        algo=config.sentence_id_algo,
        nhex=config.sentence_id_nhex,
    )

    meta = dict(g_slim.get("meta", {}))
    meta["run"] = {
        "concept": concept,
        "pipeline_version": config.pipeline_version,
        "seed": config.seed,
        "restrict_evidence_to_concept_contexts": config.restrict_evidence_to_concept_contexts,
        "corpus_stats": {"n_docs": len(corpus_docs), "n_sentences": int(sum(len(d) for d in corpus_docs))},
        "files": {
            "graph": config.graph_filename,
            "sentence_table": config.sentence_table_filename,
            "report": config.report_filename,
        },
        "timing_ms": {
            "build_art": int(t2 - t1),
            "export_graph": int(t3 - t2),
            "slim": int(t4 - t3),
            "sentence_table": int(t5 - t4),
            "total": int(t5 - t0),
        },
    }
    g_slim["meta"] = meta

    graph_path = os.path.join(config.out_dir, config.graph_filename)
    table_path = os.path.join(config.out_dir, config.sentence_table_filename)
    report_path = os.path.join(config.out_dir, config.report_filename)

    save_json(graph_path, g_slim)
    save_json(table_path, sent_table)

    graph_summary = summarize_graph(g_slim)
    report = {
        "concept": concept,
        "out_dir": config.out_dir,
        "files": meta["run"]["files"],
        "graph_summary": graph_summary,
        "timing_ms": meta["run"]["timing_ms"],
        "config": asdict(config),
        "graph_meta_digest": _short_hash(json.dumps(g_slim.get("meta", {}), ensure_ascii=False)),
    }
    save_json(report_path, report)

    return report
```

```python
def build_adjacency(graph_obj: Dict[str, Any]) -> Tuple[
    Dict[int, List[Tuple[int, Dict[str, Any]]]],
    Dict[int, List[Tuple[int, Dict[str, Any]]]],
]:
    """
    Returns out_adj, in_adj maps.
    out_adj[i] = [(j, edge_dict), ...]
    in_adj[j]  = [(i, edge_dict), ...]
    """
    out_adj: Dict[int, List[Tuple[int, Dict[str, Any]]]] = {}
    in_adj: Dict[int, List[Tuple[int, Dict[str, Any]]]] = {}
    for e in graph_obj.get("edges", []):
        i = int(e["source"])
        j = int(e["target"])
        out_adj.setdefault(i, []).append((j, e))
        in_adj.setdefault(j, []).append((i, e))
    return out_adj, in_adj
```

```python
from __future__ import annotations

from typing import Any, Dict


def summarize_graph(graph_obj: Dict[str, Any]) -> Dict[str, Any]:
    edges = graph_obj.get("edges", []) or []
    nodes = graph_obj.get("nodes", []) or []

    hist: Dict[str, int] = {}
    for e in edges:
        t = str(e.get("edge_type", "unsure"))
        hist[t] = hist.get(t, 0) + 1

    ambig = hist.get("mixed", 0) + hist.get("unsure", 0)
    total = len(edges)
    return {
        "n_nodes": len(nodes),
        "n_edges": total,
        "edge_type_hist": dict(sorted(hist.items(), key=lambda kv: (-kv[1], kv[0]))),
        "ambiguous_edges": int(ambig),
        "ambiguous_ratio": float(ambig / total) if total else 0.0,
    }
```

```python
def save_json(path: str, obj: Any) -> None:
    with open(path, "w", encoding="utf-8") as f:
        json.dump(obj, f, ensure_ascii=False, indent=2)
```

```python
from __future__ import annotations

from typing import List, Sequence, Union


def normalize_corpus_docs(
    corpus: Union[Sequence[str], Sequence[Sequence[str]]],
    *,
    drop_empty: bool = True,
    strip: bool = True,
) -> List[List[str]]:
    if not corpus:
        return []

    if isinstance(corpus[0], str):  # type: ignore[index]
        docs = [list(corpus)]  # type: ignore[arg-type]
    else:
        docs = [list(doc) for doc in corpus]  # type: ignore[assignment]

    out: List[List[str]] = []
    for doc in docs:
        cleaned: List[str] = []
        for s in doc:
            if s is None:
                continue
            ss = str(s)
            if strip:
                ss = ss.strip()
            if drop_empty and not ss:
                continue
            cleaned.append(ss)
        out.append(cleaned)
    return out


def load_corpus_from_txt(path: str) -> List[str]:
    """
    One sentence per line.
    """
    with open(path, "r", encoding="utf-8") as f:
        lines = [ln.strip() for ln in f.readlines()]
    return [ln for ln in lines if ln]
```

```python
from __future__ import annotations

from typing import Any, Dict, Sequence
import hashlib


def sentence_id(sentence: str, *, algo: str = "sha1", nhex: int = 16) -> str:
    h = hashlib.new(algo)
    h.update(sentence.encode("utf-8", errors="ignore"))
    return h.hexdigest()[:nhex]


def build_sentence_table_from_corpus(
    corpus_docs: Sequence[Sequence[str]],
    *,
    algo: str = "sha1",
    nhex: int = 16,
    dedupe: bool = True,
) -> Dict[str, str]:
    """
    Build {sentence_id: sentence}
    """
    table: Dict[str, str] = {}
    for doc in corpus_docs:
        for s in doc:
            sid = sentence_id(s, algo=algo, nhex=nhex)
            if dedupe:
                table.setdefault(sid, s)
            else:
                table[sid] = s
    return table


def attach_sentence_table_reference(
    graph_obj: Dict[str, Any],
    *,
    table_name: str = "sentence_table.json",
    algo: str = "sha1",
    nhex: int = 16,
) -> Dict[str, Any]:
    out = dict(graph_obj)
    meta = dict(out.get("meta", {}))
    meta["sentence_table_ref"] = {"filename": table_name, "algo": algo, "nhex": int(nhex), "key": "sentence_id"}
    out["meta"] = meta
    return out
```
