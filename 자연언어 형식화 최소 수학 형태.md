아래는 **가장 단순하지만 충분히 강한** 선택입니다.

# 0. 객체 확정

- 하층 상태: (x_t \in \mathbb{R}^d) (임베딩 벡터)
    
- 상층 상태: (R_t) = 가중 관계 그래프의 가중치 텐서(또는 행렬) + 제약집합  
    우선 MVP에서는 **가중치만**으로 시작합니다. (제약은 2단계)
    

관계 타입 (\tau \in {1,\dots,T}) 를 두면,  
[  
W_t^{(\tau)} \in \mathbb{R}^{N\times N}, \quad R_t \equiv {W_t^{(\tau)}}_{\tau=1}^T  
]  
여기서 (N)은 “개념 노드”의 수(초기에는 작은 사전으로 제한).

---

# 1. 하층 (f) 최소 형태 확정: 선형-수축(Contractive) 동역학

## 1-1. 형태

[  
x_{t+1} = f(x_t) = A x_t + b  
]

여기서 **수렴(응축)**을 보장하려면 최소로 이것만 추가하면 됩니다.

> (\rho(A) < 1) (스펙트럴 반경 < 1) 또는 (|A|_2 \le 1-\epsilon)

즉, (A)를 “수축 사상”으로 강제.

---

## 1-2. 학습 데이터(전이쌍) 정의를 수학적으로 확정

전이쌍은 이렇게 만듭니다.

- 컨텍스트(문장/짧은 문단) (c)
    
- 다음 컨텍스트 후보 집합 (\mathcal{N}(c)) 는 이전에 확정한 `propose_next_contexts` 결과
    
- 임베딩 (\mathrm{E}(\cdot))
    

[  
x = \mathrm{E}(c),\quad \mathcal{X}^+ = {\mathrm{E}(c') : c'\in \mathcal{N}(c)}  
]

“다음 상태”는 MVP에서는 **평균**으로 잡습니다(가장 안정적).

[  
y := \frac{1}{|\mathcal{X}^+|}\sum_{x^+ \in \mathcal{X}^+} x^+  
]

그래서 데이터셋:  
[  
\mathcal{D} = {(x_i, y_i)}_{i=1}^m  
]

---

## 1-3. (A,b) 추정식을 확정 (릿지 회귀 + 수축 강제)

### 기본(폐형식) 해

릿지 회귀로  
[  
\min_{A,b}\sum_{i=1}^m |Ax_i + b - y_i|_2^2 + \lambda |A|_F^2  
]

편의를 위해 (b)를 포함한 확장 벡터를 쓰면 닫힌 형태가 나옵니다.

- (x_i' := \begin{bmatrix} x_i \ 1 \end{bmatrix}\in\mathbb{R}^{d+1})
    
- (B := [A;; b] \in \mathbb{R}^{d\times(d+1)})
    

[  
\min_B |BX' - Y|_F^2 + \lambda |B|_F^2  
]

여기서

- (X' = [x_1',...,x_m'])
    
- (Y = [y_1,...,y_m])
    

해는:  
[  
B^* = Y X'^T \left(X'X'^T + \lambda I\right)^{-1}  
]

### 수축 강제(최소 조작)

학습 후 (A)에 대해 스케일링 한 번만 합니다.

1. (A)의 스펙트럴 노름(또는 최대 특이값) (\sigma_{\max}(A)) 계산
    
2. 만약 (\sigma_{\max}(A) > 1-\epsilon) 이면  
    [  
    A \leftarrow \frac{1-\epsilon}{\sigma_{\max}(A)}A  
    ]
    

이걸로 **수렴 조건을 강제**합니다. (MVP에 가장 실용적)

# 2. 상층 (\Phi) 최소 형태 확정: 가중치 강화/망각 + 가지치기

상층은 “관계 가중치가 의미 상태 (x_t)에 의해 업데이트된다” 입니다.

## 2-1. (R)를 최소로 단순화

초기 MVP는 관계 타입 하나만 둡니다((T=1)).

[  
R_t \equiv W_t \in \mathbb{R}^{N\times N}  
]

(관계 타입 분리는 다음 단계에서.)

---

## 2-2. (\Phi) 형태

[  
W_{t+1} = \Phi(W_t, x_t) = (1-\eta)W_t + \eta, S(x_t)  
]

- (\eta\in(0,1)): 관성/학습률
    
- (S(x)): 현재 의미 상태 (x)가 “지지하는 관계”를 바로 찍어내는 **스코어 행렬**
    

이 한 줄이면 끝입니다.

---

## 2-3. (S(x))를 “학습 없이” 확정 (규칙 기반 최소형)

핵심은: (S(x))를 데이터에서 뽑되, 학습모델 없이도 돌아가게 만들기.

### (a) 후보 노드(개념) 세트

개념 노드 (j)마다 “대표 벡터” (u_j \in \mathbb{R}^d) 를 미리 둡니다.  
(방법: 그 개념의 코어 컨텍스트 임베딩 평균)

[  
u_j = \mathrm{centroid}({\mathrm{E}(c): c \text{ in core contexts of concept } j})  
]

### (b) 현재 상태 (x)에서 각 노드의 활성도(관련도)

[  
a_j(x) = \mathrm{softmax}\left(\beta, \cos(x, u_j)\right)  
]  
또는 단순히 (\cos(x,u_j))를 쓰되, MVP에서는 softmax가 안정적입니다.

### (c) 관계 스코어 행렬 (S(x))

가장 단순한 관계 생성은 “동시 활성”입니다.

[  
S_{ij}(x) = a_i(x),a_j(x)  
]

즉, 현재 의미상태에서 동시에 켜지는 개념 쌍이 관계가 강해집니다.  
이게 **가족유사성 의미망**을 가장 직접적으로 반영합니다.

> 한 줄로 말하면: “함께 떠오르는 것끼리 엣지를 강화한다.”

---

## 2-4. 가지치기(pruning) 확정

꼬리 영역을 버리는 규칙을 수학적으로 박습니다.

[  
W_{t+1,ij} \leftarrow 0 \quad \text{if } W_{t+1,ij} < \theta  
]

또는 상위 (k)개만 남김:

[  
W_{t+1,i,\cdot} \text{ keep top-}k  
]

MVP는 임계치 (\theta)가 더 단순합니다.

---

# 3. 전체 반복(롤아웃)과 “응축” 판정 확정

반복:  
[  
x_{t+1}=Ax_t+b,\qquad W_{t+1}=(1-\eta)W_t+\eta S(x_t)  
]

응축 판정:

- 하층:  
    [  
    |x_{t+1}-x_t|_2 \le \varepsilon_x  
    ]
    
- 상층:  
    [  
    |W_{t+1}-W_t|_F \le \varepsilon_W  
    ]
    

동시에 만족하면 수렴.

---

# 4. 형식화 결과물(최소 출력) 확정

개념 (C)의 형식화 결과는

[  
\boxed{C ;\mapsto; (A,b;\ \eta,\beta,\theta;\ {u_j}_{j=1}^N)}  
]

즉

- 하층 생성 규칙: (A,b)
    
- 상층 업데이트 규칙: (\eta) + 활성도 온도 (\beta) + 가지치기 (\theta)
    
- 개념 노드 프로토타입: (u_j)
    

이렇게 **완전히 연산 가능한 형태**로 닫힙니다.

---

# 5. 이 최소 형태가 주는 실질적 효과

- 의미공간이 무한히 열려 있어도  
    (a_j(x))가 사실상 중심부에서만 크게 나오고  
    pruning((\theta))이 꼬리를 자르므로  
    계산이 안정됩니다.
    
- (A)를 수축으로 강제했기 때문에  
    “의미 상태가 발산해서 형식화 실패”를 기본적으로 막습니다.
